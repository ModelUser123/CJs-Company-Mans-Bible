## Cased vs. Open Hole Decision

```
              ┌───────────────────────────────┐

│                                                           │

         ┌────────────────────┴────────────────────┐
```

### Sand Control Method Details

```
┌─────────────────────────────────────────────────────────────┐
│                STANDALONE SCREENS                           │
│                                                             │
│ BEST FOR:                                                  │
│ • High permeability formations                             │
│ • Competent sand                                           │
│ • Horizontal wells                                         │
│ • Lower cost requirement                                   │
│                                                             │
│ TYPES:                                                     │
│ • Wire wrapped screens                                     │
│ • Premium mesh screens                                     │
│ • Expandable screens                                       │
│                                                             │
│ LIMITATIONS:                                               │
│ • No annular pack (plugging risk)                         │
│ • Fines migration concern                                  │
│ • Limited life in poorly consolidated sand                │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                   GRAVEL PACK                               │
│                                                             │
│ BEST FOR:                                                  │
│ • Unconsolidated formations                                │
│ • High production rates needed                             │
│ • Long-term sand exclusion required                        │
│                                                             │
│ TYPES:                                                     │
│ • Cased hole gravel pack (CHGP)                           │
│ • Open hole gravel pack (OHGP)                            │
│ • High-rate water pack (HRWP)                             │
│                                                             │
│ CONSIDERATIONS:                                            │
│ • Proper gravel sizing critical                           │
│ • Complete packing essential                              │
│ • Higher cost than screens alone                          │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│                    FRAC PACK                                │
│                                                             │
│ BEST FOR:                                                  │
│ • Medium to low permeability                               │
│ • Combined stimulation + sand control                     │
│ • Bypasses near-wellbore damage                           │
│                                                             │
│ ADVANTAGES:                                                │
│ • Creates high-conductivity path                          │
│ • Controls sand production                                 │
│ • Improves well productivity                              │
│                                                             │
│ CONSIDERATIONS:                                            │
│ • Higher cost and complexity                               │
│ • Requires frac equipment                                  │
│ • TSO (tip screen out) technique                          │
└─────────────────────────────────────────────────────────────┘
```

## Completion Selection Summary

| Factor | Recommendation |
|--------|---------------|
| High perm, no sand | Open hole or simple perf |
| High perm, sand | Standalone screen or gravel pack |
| Med perm, sand | Frac pack |
| Low perm | Hydraulic fracturing |
| Multiple zones, compatible | Commingled |
| Multiple zones, incompatible | Selective with isolation |
| High rate, deep | ESP |
| Low rate, shallow | Rod pump |
| High GOR | Gas lift |
| Heavy oil | PCP |

---

*Completion design should optimize long-term well performance while considering intervention requirements and cost.*
