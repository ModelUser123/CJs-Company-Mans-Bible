### High Pressure During Cementing

```
┌─────────────────────────────────────────────────────────────┐
│              HIGH PRESSURE DURING PUMPING                   │
│        (Above expected/predicted pressures)                │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
              ┌───────────────────────────────┐
              │   WHERE IN THE JOB?           │
              └───────────────────────────────┘
         │                   │                   │
         ▼                   ▼                   ▼
┌───────────────┐   ┌───────────────┐   ┌───────────────┐
│ EARLY         │   │ MID-JOB       │   │ NEAR END      │
│ (Spacer/wash) │   │ (Lead cement) │   │ (Tail/bump)   │
└───────────────┘   └───────────────┘   └───────────────┘
         │                   │                   │
         ▼                   ▼                   ▼

EARLY (SPACER/WASH):
┌─────────────────────────────────────────────────────────────┐
│ POSSIBLE CAUSES:                                           │
│ • Float equipment plugged                                  │
│ • Settled solids in casing                                │
│ • Incompatible spacer/mud                                  │
│                                                             │
│ ACTIONS:                                                   │
│ 1. Reduce rate, attempt to break loose                    │
│ 2. Apply pressure in steps (500 psi increments)           │
│ 3. If blocked, may need to POOH and redress               │
│ 4. DO NOT exceed casing burst or frac gradient            │
└─────────────────────────────────────────────────────────────┘

MID-JOB (LEAD CEMENT):
┌─────────────────────────────────────────────────────────────┐
│ POSSIBLE CAUSES:                                           │
│ • Bridging in annulus                                      │
│ • Cement flash setting                                     │
│ • Underreamed pocket filling                              │
│ • Lost circulation (U-tubing)                             │
│                                                             │
│ ACTIONS:                                                   │
│ 1. Reduce pump rate                                        │
│ 2. Check for returns                                       │
│ 3. If pressure continues high, may need to stop           │
│ 4. Evaluate cement coverage vs. continuing                │
└─────────────────────────────────────────────────────────────┘

NEAR END (TAIL/BUMP):
┌─────────────────────────────────────────────────────────────┐
│ LIKELY CAUSE: Reaching plug bump                          │
│                                                             │
│ VERIFY:                                                    │
│ • Check displacement volume pumped                         │
│ • Should be at expected bump volume                        │
│ • Pressure should be steady when stopped                   │
│                                                             │
│ IF PREMATURE:                                              │
│ • Cement may have channeled past plug                     │
│ • Float may have closed early                              │
│ • Plug may have set early                                  │
└─────────────────────────────────────────────────────────────┘
```

### Lost Returns During Cementing

```
┌─────────────────────────────────────────────────────────────┐
│           LOST RETURNS DURING CEMENTING                     │
│         (Partial or complete loss of returns)              │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│               IMMEDIATE ASSESSMENT                          │
│                                                             │
│ • At what volume did losses occur?                        │
│ • Is cement still inside casing or in annulus?            │
│ • Can you continue pumping?                                │
│ • What is the rate of loss?                                │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
              ┌───────────────────────────────┐
              │   IS CEMENT IN ANNULUS?       │
              └───────────────────────────────┘
                     │              │
                 YES │              │ NO (still in casing)
                     ▼              ▼
┌─────────────────────────────────────────────────────────────┐
│ CEMENT IN ANNULUS:                                         │
│                                                             │
│ OPTIONS:                                                   │
│ 1. Continue pumping if cement will reach target height    │
│ 2. Accept partial fill and plan remedial cement           │
│ 3. Increase rate to outrun losses (risky)                 │
│                                                             │
│ DO NOT:                                                    │
│ • Stop with cement in casing (will set around drillpipe)  │
│ • Exceed frac gradient trying to force cement up          │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│ IF LOSSES ARE SEVERE/TOTAL:                                │
│                                                             │
│ 1. Continue pumping tail cement to protect shoe           │
│ 2. Displace to plug bump if possible                      │
│ 3. Wait on cement (WOC)                                    │
│ 4. Evaluate cement top with CBL/VDL                       │
│ 5. Plan remedial (top-up) cement job if needed            │
└─────────────────────────────────────────────────────────────┘
```

### No Bump Achieved

```
┌─────────────────────────────────────────────────────────────┐
│               NO BUMP ACHIEVED                              │
│      (Calculated volume pumped, no pressure increase)      │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│               POSSIBLE CAUSES                               │
│                                                             │
│ 1. Plug(s) destroyed during pumping                        │
│ 2. Plug bypassed (slotted liner, holes in casing)         │
│ 3. Plug still in casing (not enough displacement)         │
│ 4. Float equipment failure                                 │
│ 5. Casing leak/failure                                     │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│               ACTIONS                                       │
│                                                             │
│ 1. Continue pumping additional 10-20 bbl                   │
│ 2. Monitor pressure and returns                            │
│ 3. If no bump after +20 bbl, stop                         │
│ 4. Wait on cement                                          │
│ 5. Run bond log to evaluate cement                        │
│ 6. May need remedial cementing                            │
│                                                             │
│ DO NOT over-displace (will U-tube cement back)            │
└─────────────────────────────────────────────────────────────┘
```

## Cement Job Quick Reference

| Problem | During Job Action | Post-Job Action |
|---------|------------------|-----------------|
| High pressure | Reduce rate, check plugging | Evaluate, drill out |
| Lost returns | Continue if cement in annulus | Top-up cement |
| Pump failure | Switch backup, keep moving | Evaluate position |
| No bump | Pump 10-20 bbl extra | CBL, remedial if needed |
| Premature bump | Verify volume | Check for channel |
| Float won't hold | Hold pressure on casing | CBL, may need squeeze |

---

*A good cement job starts with good preparation. Take time to verify all parameters before pumping.*
