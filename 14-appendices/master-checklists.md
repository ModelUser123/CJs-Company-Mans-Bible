# Master Checklists

## Pre-Spud Checklist

### Documentation Review
- [ ] AFE approved and on location
- [ ] Drilling permit approved (APD/state)
- [ ] Surface use agreement signed
- [ ] Road use agreement in place
- [ ] Water rights/permits secured
- [ ] Survey plat verified
- [ ] Well prognosis reviewed
- [ ] Casing program confirmed
- [ ] Mud program approved
- [ ] Drilling contracts signed
- [ ] Service company contracts in place
- [ ] Insurance certificates verified
- [ ] Emergency response plan on location

### Location Preparation
- [ ] Pad construction complete
- [ ] Rig access road adequate
- [ ] Cellar dug to specifications
- [ ] Conductor installed (if required)
- [ ] Mouse/rat holes drilled
- [ ] Rig electrical/water/sewer connected
- [ ] Fuel supply arranged
- [ ] Communication systems tested
- [ ] Security measures in place
- [ ] Location signs posted

### Rig Inspection
- [ ] Rig inspection completed
- [ ] Third-party inspection (if required)
- [ ] BOP function tested
- [ ] BOP pressure tested
- [ ] Accumulator precharge verified
- [ ] Choke manifold tested
- [ ] Mud system clean
- [ ] Shale shaker screens available
- [ ] Standpipe/kelly hose tested
- [ ] Drawworks brakes tested
- [ ] Top drive/kelly in good condition
- [ ] Drill line inspected/slip-cut

### Safety Systems
- [ ] H2S monitors calibrated
- [ ] Fire extinguishers inspected
- [ ] Eye wash stations functional
- [ ] First aid kit stocked
- [ ] Safety shower functional
- [ ] PPE available for all personnel
- [ ] MSDS/SDS sheets on location
- [ ] Evacuation routes posted
- [ ] Muster point identified
- [ ] Emergency contact list posted

## Trip Checklist

### Pre-Trip Preparation
- [ ] Hole in good condition
- [ ] Circulated bottoms up
- [ ] Mud properties checked
- [ ] Pump strokes calibrated
- [ ] Fill-up pump ready
- [ ] Trip tank calibrated
- [ ] Slug calculated and mixed (if needed)
- [ ] Crew briefed on expected conditions

### During Trip Out
- [ ] Trip sheet started
- [ ] Fill volume monitored continuously
- [ ] Swab pressure considered (speed control)
- [ ] Flow checks performed per protocol
- [ ] Bit condition noted when at surface
- [ ] BHA condition inspected
- [ ] Pipe visually inspected

### During Trip In
- [ ] Pipe count verified
- [ ] Pipe measured
- [ ] Displacement monitored
- [ ] Surge pressure considered (speed control)
- [ ] Tag bottom carefully
- [ ] Circulate and condition mud
- [ ] Take survey (if directional)

### Post-Trip
- [ ] Trip sheet complete
- [ ] Volumes balanced
- [ ] Any anomalies documented
- [ ] Mud report updated
- [ ] Daily report completed

## Casing Running Checklist

### Pre-Job
- [ ] Casing tallied and verified
- [ ] Thread protectors removed, threads inspected
- [ ] Drift all casing
- [ ] Centralizers ready
- [ ] Float equipment ready
- [ ] Cement program reviewed
- [ ] Casing accessories on location
- [ ] Tongs/power tongs tested
- [ ] Thread compound available
- [ ] Running speed calculated
- [ ] Makeup torque values posted
- [ ] Crew trained on procedure

### Running Casing
- [ ] Fill casing while running
- [ ] Stab and makeup carefully
- [ ] Apply thread compound properly
- [ ] Torque monitor all connections
- [ ] Install centralizers per program
- [ ] Record joint count and depths
- [ ] Monitor for flow (well control)
- [ ] Circulate/rotate if stuck

### Landing Casing
- [ ] Land casing at proper depth
- [ ] Check weight - land or hang off
- [ ] Lockdown ring (if required)
- [ ] Test float equipment
- [ ] Circulate for cement

## BOP Test Checklist

### Function Test
- [ ] Annular - open and close
- [ ] Upper pipe rams - open and close
- [ ] Lower pipe rams - open and close
- [ ] Blind/shear rams - open and close
- [ ] Position indicators verified
- [ ] Time to close recorded
- [ ] Remote panel function verified
- [ ] Backup system function verified

### Low Pressure Test
- [ ] Test tool installed
- [ ] Pressure up to 200-300 psi
- [ ] Hold for 5 minutes
- [ ] Zero pressure drop acceptable
- [ ] Test each preventer individually
- [ ] Test choke line
- [ ] Test kill line

### High Pressure Test
- [ ] Pressure up to 70% working pressure
- [ ] Hold for 5 minutes
- [ ] Zero pressure drop acceptable
- [ ] Test each preventer individually
- [ ] Test choke manifold
- [ ] Test all surface connections
- [ ] Document all results

## Well Kill Checklist

### Initial Response
- [ ] Recognize kick indicators
- [ ] Stop drilling/tripping
- [ ] Pick up off bottom
- [ ] Start pumps (if drilling)
- [ ] Observe flow
- [ ] Shut-in well (hard or soft)
- [ ] Record pressures and time

### After Shut-In
- [ ] Record stabilized SIDPP
- [ ] Record stabilized SICP
- [ ] Calculate pit gain
- [ ] Notify supervision
- [ ] Calculate kill mud weight
- [ ] Verify slow pump rates
- [ ] Select kill method
- [ ] Brief crew

### During Kill
- [ ] Follow kill sheet
- [ ] Monitor pressures continuously
- [ ] Adjust choke to maintain BHP
- [ ] Track pump strokes
- [ ] Monitor pit volumes
- [ ] Watch for complications
- [ ] Document everything

### Post-Kill
- [ ] Verify well dead (flow check)
- [ ] Increase mud weight if needed
- [ ] Resume operations cautiously
- [ ] Investigate kick source
- [ ] Complete incident report
- [ ] Notify as required

## Daily Operations Checklist

### Shift Start
- [ ] Read previous shift reports
- [ ] Review current status
- [ ] Check mud properties
- [ ] Verify well volumes
- [ ] Inspect BOP status lights
- [ ] Review upcoming operations
- [ ] Brief crew if needed

### Throughout Shift
- [ ] Monitor drilling parameters
- [ ] Watch for abnormalities
- [ ] Maintain communications
- [ ] Document all activities
- [ ] Conduct safety observations
- [ ] Monitor contractors

### Shift End
- [ ] Complete daily report
- [ ] Update tally/depth
- [ ] Note any issues
- [ ] Brief relief
- [ ] Sign off reports

---

*Checklists should be customized for specific operations and company requirements. This provides a foundation.*
