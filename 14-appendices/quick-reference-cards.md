## CARD 3: Common Drilling Calculations

```
┌─────────────────────────────────────────────────────────────────┐
│                   DRILLING CALCULATIONS CARD                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  HYDROSTATIC PRESSURE:                                          │
│  HP (psi) = MW (ppg) × 0.052 × TVD (ft)                         │
│                                                                 │
│  EQUIVALENT MUD WEIGHT:                                         │
│  EMW (ppg) = Pressure (psi) ÷ 0.052 ÷ TVD (ft)                  │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  PIPE CAPACITY:                                                 │
│  Cap (bbl/ft) = ID² ÷ 1029.4                                    │
│                                                                 │
│  ANNULAR CAPACITY:                                              │
│  Cap (bbl/ft) = (Hole² - Pipe²) ÷ 1029.4                        │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  BOTTOMS UP STROKES:                                            │
│  Strokes = Annular Volume (bbl) ÷ Pump Output (bbl/stk)         │
│                                                                 │
│  BOTTOMS UP TIME:                                               │
│  Time (min) = Strokes ÷ SPM                                     │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ANNULAR VELOCITY:                                              │
│  AV (ft/min) = 24.5 × Q (gpm) ÷ (Hole² - Pipe²)                 │
│  Target: >100 ft/min vertical, >150 ft/min horizontal           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
