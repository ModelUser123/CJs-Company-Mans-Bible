## Temperature Conversions

### Temperature Formulas

```
FAHRENHEIT TO CELSIUS
°C = (°F - 32) × 5/9

CELSIUS TO FAHRENHEIT
°F = (°C × 9/5) + 32

FAHRENHEIT TO RANKINE
°R = °F + 459.67

CELSIUS TO KELVIN
K = °C + 273.15
```

## Common Oilfield Formulas

### Hydrostatic Pressure
```
HP (psi) = MW (ppg) × 0.052 × TVD (ft)
```

### Annular Volume
```
AV (bbl/ft) = (D₁² - D₂²) / 1029.4
Where: D₁ = hole OD (in), D₂ = pipe OD (in)
```

### Pipe Capacity
```
Cap (bbl/ft) = ID² / 1029.4
```

### Pump Output
```
Output (bbl/stroke) = (D² × L × Eff) / 6,310.3
Where: D = liner diameter (in), L = stroke length (in)
```

### ECD
```
ECD (ppg) = MW + (ΔP_annular / 0.052 / TVD)
```
