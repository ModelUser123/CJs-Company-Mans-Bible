# Fishing Tools Reference

## Engagement Tools

### Overshots

```
OVERSHOT DESIGN
├── Purpose: Engage over outside of fish
├── Components
│   ├── Top Sub (connection to string)
│   ├── Bowl (outer body)
│   ├── Grapple (gripping element)
│   │   ├── Spiral type (most common)
│   │   ├── Basket type (damaged fish)
│   │   └── Pack-off type (sealing)
│   ├── Guide (centers over fish)
│   └── Control (release mechanism)
└── Operation
    ├── Lower over fish top
    ├── Grapple engages with downward motion
    ├── Pull up to tighten grip
    └── Release with rotation (if stuck)

OVERSHOT SIZING
├── Catch Range
│   ├── Full size: Catches specific OD
│   ├── Oversize: Catches range of ODs
│   └── Must match fish OD within range
├── Bowl Size Selection
│   ├── Bowl ID > Fish OD + clearance
│   ├── Bowl OD < Hole ID
│   └── Sufficient wall for strength
└── Example
    ├── 5" fish OD
    ├── 6-1/8" grapple
    ├── 7-3/4" bowl
    └── For 8-1/2" hole
```

### Overshot Types and Applications

| Type | Catch Range | Application |
|------|-------------|-------------|
| Standard spiral grapple | Specific OD | Normal fishing |
| Basket grapple | Range of ODs | Damaged fish top |
| Pack-off grapple | Specific OD | Circulating/washing |
| Releasing overshot | Specific OD | Uncertain engagement |

### Spears

```
SPEAR DESIGN
├── Purpose: Engage inside fish ID
├── Components
│   ├── Top Sub (connection)
│   ├── Mandrel (body)
│   ├── Slip/Grapple assembly
│   │   ├── Releasing (can be reset)
│   │   └── Non-releasing (stronger grip)
│   └── Catch ring (depth control)
└── Operation
    ├── Enter fish bore
    ├── Set depth (catch ring contacts)
    ├── Pull up to engage slips
    └── Release with down weight + rotation

SPEAR SIZING
├── Spear OD < Fish ID
├── Slip expansion must match fish ID
├── Mandrel strong enough for pull
└── Example
    ├── Fish ID = 3.826" (4-1/2" casing)
    ├── Spear = 3.70" slip OD
    └── Expansion range covers 3.826"
```

### Spear Types

| Type | Feature | Application |
|------|---------|-------------|
| Releasing | Can reset/release | Exploratory, uncertain fish |
| Non-releasing | Maximum grip | Known good fish |
| Boot sub spear | Has guide sub | Off-center fish |
| Bumper sub spear | Integral jar | Limited tool space |

### Taper Taps

```
TAPER TAP
├── Purpose: Thread into damaged box connection
├── Design
│   ├── Tapered threading
│   ├── Matching thread form
│   └── Hardened cutting surface
├── Operation
│   ├── Rotate clockwise into fish
│   ├── Cuts new threads as it enters
│   ├── Engages box connection
│   └── Pull to retrieve
└── Limitations
    ├── Damages fish connection
    ├── Works only on box (female)
    ├── Requires rotation
    └── May not hold on hard pull
```

### Die Collars

```
DIE COLLAR
├── Purpose: Thread onto damaged pin connection
├── Design
│   ├── Internal tapered threads
│   ├── Matching thread form
│   └── Hardened cutting surface
├── Operation
│   ├── Lower over fish pin
│   ├── Rotate to cut/engage threads
│   ├── Pull to retrieve
└── Similar limitations to taper tap
```

## Jarring Tools

### Hydraulic Jars

```
HYDRAULIC JAR
├── Purpose: Deliver impact force to free stuck fish
├── Operation
│   ├── Up Jar: Pull creates upward impact
│   ├── Down Jar: Slack creates downward impact
│   └── Bidirectional: Both directions
├── Mechanism
│   ├── Metering restricts hydraulic flow
│   ├── Sudden release creates impact
│   ├── Force increases with overpull/slack
│   └── Time delay allows cocking
├── Sizing
│   ├── Match to string size
│   ├── Output in ft-lbs or kJ
│   └── Temperature rating
└── Settings
    ├── Pre-load adjustment
    ├── Metering rate
    └── Stroke length

HYDRAULIC JAR OUTPUT (Typical)
├── Light: 10,000-25,000 ft-lb
├── Medium: 25,000-50,000 ft-lb
├── Heavy: 50,000-100,000+ ft-lb
└── Application depends on fish/stuck condition
```

### Mechanical Jars

```
MECHANICAL JAR
├── Purpose: Deliver impact without hydraulics
├── Operation
│   ├── Cocking: Apply overpull or slack
│   ├── Trigger: Release mechanism fires
│   ├── Impact: Mass accelerates
│   └── Reset: Re-cock for next blow
├── Types
│   ├── Trip-release (tension triggered)
│   ├── Compression-release
│   └── Rotary-release
├── Advantages
│   ├── Works at higher temperatures
│   ├── More consistent in high angle
│   ├── Simpler mechanism
│   └── Less sensitive to fluid
└── Disadvantages
    ├── Generally lower output
    ├── Preset trigger load
    └── More wear-prone
```

### Bumper Subs & Accelerators

```
BUMPER SUB
├── Purpose: Allow string movement for jar operation
├── Design
│   ├── Telescoping sections
│   ├── Stroke length: 8-18 inches typical
│   └── Mandrel slides in housing
├── Function
│   ├── Absorbs movement for jar cocking
│   ├── Allows jar to fully extend/stroke
│   └── Prevents string from limiting jar action
└── Placement: Above or below jar

ACCELERATOR
├── Purpose: Increase jar impact force
├── Design
│   ├── Sealed chamber with gas or spring
│   ├── Pre-charged with nitrogen
│   └── Stores energy during cocking
├── Function
│   ├── Additional stored energy
│   ├── Accelerates jar mandrel
│   └── Significantly increases impact
└── Can multiply jar output 2-4×
```

## Milling Tools

### Junk Mills

```
JUNK MILL TYPES
├── Flat Bottom Mill
│   ├── Grinds flat surfaces
│   ├── Used for most junk
│   └── Standard first choice
├── Concave Mill
│   ├── Dished bottom
│   ├── Centers on fish
│   └── Good for round objects
├── Convex Mill
│   ├── Rounded bottom
│   ├── Pilots into fish
│   └── Starting casing mills
├── Taper Mill
│   ├── Tapered OD
│   ├── Opens restrictions
│   └── Dresses damaged casing
└── Pilot Mill
    ├── Small center pilot
    ├── Keeps mill centered
    └── Prevents wallowing
```

### Mill Dressing

| Dressing Type | Application | Hardness |
|---------------|-------------|----------|
| Tungsten carbide | General purpose | Hard |
| Crushed carbide | Soft formations, junk | Medium |
| Diamond | Hard formations, milling casing | Very hard |
| Carbide insert | Heavy duty | Hard |

### Milling Parameters

```
MILLING GUIDELINES
├── Weight on Mill
│   ├── Light: 2,000-5,000 lb (starting)
│   ├── Medium: 5,000-10,000 lb (normal)
│   └── Heavy: 10,000-20,000 lb (hard material)
├── Rotation Speed
│   ├── Slow: 40-60 RPM (large mills)
│   ├── Medium: 60-100 RPM (standard)
│   └── Fast: 100-150 RPM (small mills)
├── Flow Rate
│   ├── Adequate for cooling and cleaning
│   ├── Typically higher than drilling
│   └── Monitor returns for cuttings
└── Progress Indicators
    ├── Torque patterns
    ├── Cuttings at surface
    ├── Weight decrease
    └── Depth progress
```

## Junk Recovery Tools

```
JUNK BASKET TYPES
├── Core-Type Junk Basket
│   ├── Fingers retain junk
│   ├── Circulating fluid helps
│   ├── For small debris
│   └── Run above bit or mill
├── Boot Basket
│   ├── Reverse circulation type
│   ├── Collected in basket
│   ├── For larger pieces
│   └── More positive recovery
├── Poor Boy Basket
│   ├── Simple finger design
│   ├── Inexpensive
│   ├── Light duty
│   └── Small junk only
└── Hydrostatic Junk Basket
    ├── Uses differential pressure
    ├── Positive seal on junk
    └── Best retention
```

### Fishing Magnets

```
FISHING MAGNET TYPES
├── Permanent Magnet

│   ├── Strong field
│   ├── Heavy
│   └── Standard choice
```

### Reverse Circulation Tools

```
REVERSE CIRCULATION JUNK BASKET
├── Purpose: Flow junk into basket
├── Operation
│   ├── Pump down annulus
│   ├── Returns up drill string
│   ├── Junk lifted into basket
│   └── Ball/dart to circulate normal
├── Best For
│   ├── Small debris
│   ├── Bit cones/inserts
│   └── Formation cuttings
└── Limitations
    ├── Size limited by ID
    ├── Requires special plumbing
    └── May not work in high angle
```

## Specialty Tools

### Washover Pipe

```
WASHOVER PIPE
├── Purpose: Pass over stuck string to free
├── Design
│   ├── Large bore pipe
│   ├── Rotary shoe on bottom
│   ├── Passes over fish
│   └── Mills away obstruction
├── Operation
│   ├── Run over fish
│   ├── Rotate and wash
│   ├── Free stuck section
│   └── Recover with conventional tools
└── Application
    ├── Differential sticking
    ├── Pack-off situations
    └── When fish can't be pulled directly
```

### String Shots (Back-Off)

```
STRING SHOT (BACK-OFF)
├── Purpose: Unscrew string at specific connection
├── Mechanism
│   ├── Explosive charge in drill string
│   ├── Detonation applies torque
│   ├── Combined with left-hand rotation
│   └── Unscrews at weak/targeted connection
├── Procedure
│   ├── Locate desired back-off point
│   ├── Apply left-hand torque
│   ├── Lower string shot to depth
│   ├── Detonate
│   └── POOH free portion
├── Applications
│   ├── Retrieve pipe above stuck point
│   ├── Controlled parting
│   └── Allows smaller fish
└── Risks
    ├── Unintended parting point
    ├── Damage to string
    └── Requires wireline/electric line
```

### Impression Blocks

```
IMPRESSION BLOCK
├── Purpose: Get imprint of fish top
├── Design
│   ├── Soft lead or aluminum base
│   ├── Collects impression when set down
│   └── Shows fish top condition
├── Use
│   ├── When fish condition unknown
│   ├── After unsuccessful engagement
│   ├── To verify fish position
│   └── Planning next tool selection
└── Interpretation
    ├── Shape shows fish configuration
    ├── Depth to fish top
    ├── Centering indication
    └── Damage/debris present
```

## Tool Selection Guide

### Quick Reference Table

| Fish Type | Primary Tool | Secondary | Notes |
|-----------|-------------|-----------|-------|
| Drill pipe (box up) | Overshot | Taper tap | Check grapple size |
| Drill pipe (pin up) | Spear | Die collar | Know fish ID |
| Drill collars | Overshot | Taper tap | Heavy pull expected |
| Damaged connection | Basket grapple | Impression block | Assess first |
| Wireline tools | Overshot | Spear | Small sizes |
| Bit cones | Magnet | Junk basket | Multiple runs |
| Small junk | Junk basket | Magnet | May need milling |
| Casing/liner | Casing spear | Overshot | High loads |
| Stuck (differential) | Washover | Spot fluid first | Time critical |

*Tool selection is critical to fishing success. Consult with experienced fishing hands and tool company representatives.*
