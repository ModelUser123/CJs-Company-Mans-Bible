# Fishing Procedures

## Pre-Fishing Preparation

### Information Gathering

```
FISH IDENTIFICATION CHECKLIST
├── What Is It?
│   ├── Component type (DP, DC, BHA, etc.)
│   ├── Size (OD, ID, length)
│   ├── Grade/specification
│   ├── Connection type
│   └── Any distinguishing features
├── Where Is It?
│   ├── Top of fish (MD)
│   ├── TVD
│   ├── Inclination/azimuth at fish
│   └── Position in hole (centered?)
├── What Condition?
│   ├── How was it created?
│   ├── Is it stuck or free?
│   ├── Top condition (damaged?)
│   └── Time since event
└── Hole Conditions
    ├── Hole size
    ├── Casing setting depths
    ├── Mud type and weight
    └── Any restrictions
```

### Tool Preparation

```
PRE-RUN TOOL CHECKS
├── Overshot/Spear
│   ├── Correct size for fish
│   ├── Grapple condition
│   ├── Release mechanism tested
│   ├── All components assembled
│   └── Catch range verified
├── Jar
│   ├── Correct size for string
│   ├── Pre-load setting correct
│   ├── Lubricated and tested
│   └── Firing mechanism functional
├── Bumper Sub
│   ├── Stroke length adequate
│   ├── Properly made up
│   └── No leaks
├── Accessories
│   ├── Safety joints functional
│   ├── Crossovers correct
│   └── All connections checked
└── Backup Tools Ready
    ├── Alternative fishing tool
    ├── Mill (if needed)
    └── Impression block
```

## Standard Fishing Procedure

### Running In Hole

```
FISHING RIH PROCEDURE
├── Safety Meeting
│   ├── Review procedure
│   ├── Discuss contingencies
│   ├── Assign responsibilities
│   └── Set time limits
├── Run Fishing String
│   ├── Proper pipe handling
│   ├── Connection makeup per spec
│   ├── Monitor weight
│   └── Control speed in build
├── Approaching Fish
│   ├── Slow 100-200 ft above fish
│   ├── Circulate and condition
│   ├── Establish baseline weight
│   └── Note any drag/tight spots
├── Tag Fish
│   ├── Lower slowly to fish
│   ├── Note tag depth
│   ├── Compare to calculated depth
│   └── Assess fish position
└── Pre-Engagement Circulation
    ├── Circulate bottoms up
    ├── Clean around fish
    └── Stable pressures/weight
```

### Engaging the Fish

#### Overshot Engagement

```
OVERSHOT ENGAGEMENT
├── Position Over Fish
│   ├── Lower overshot over fish top
│   ├── Feel for fish entering bowl
│   ├── Watch weight indicator
│   └── Grapple should pass over smoothly
├── Set Grapple
│   ├── Continue down until guide contacts
│   ├── Note depth (grapple position)
│   ├── Pull up to engage grapple
│   └── Pickup weight should increase
├── Verify Engagement
│   ├── Record weight increase
│   ├── Should feel solid
│   ├── Circulate to check seal (if pack-off)
│   └── Ready to work fish
└── If No Engagement
    ├── Raise and re-try
    ├── Check fish position
    ├── May need impression block
    └── Consider different approach
```

#### Spear Engagement

```
SPEAR ENGAGEMENT
├── Enter Fish Bore
│   ├── Lower spear into fish ID
│   ├── Watch weight indicator
│   ├── Feel for restrictions
│   └── Note depth of entry
├── Set Catch Ring
│   ├── Lower until catch ring contacts
│   ├── This sets slip position
│   ├── Critical for proper grip
│   └── Record depth
├── Engage Slips
│   ├── Pull up to set slips
│   ├── Weight should increase
│   ├── Slips expand against ID
│   └── Verify solid engagement
└── If No Engagement
    ├── Slips may be wrong size
    ├── Debris in fish bore
    ├── Need smaller/larger spear
    └── Try different depth setting
```

### Working the Fish

```
FREEING STUCK FISH
├── Initial Pull
│   ├── Apply steady overpull
│   ├── Monitor weight increase
│   ├── Note fish response
│   └── Do not exceed limits
├── If Pipe Moves
│   ├── Continue pulling
│   ├── Work up hole slowly
│   ├── Watch for hang-ups
│   └── May come free easily
├── If Pipe Doesn't Move → Jar
│   ├── Apply overpull to cock jar
│   ├── Wait for jar to fire
│   ├── Monitor for response
│   └── Repeat as needed
├── Jarring Sequence
│   ├── Start with lighter blows
│   ├── Increase force gradually
│   ├── Vary overpull amount
│   ├── Allow time between blows
│   └── Monitor for progress
└── Circulation While Stuck
    ├── Circulate if possible
    ├── Helps wash around fish
    ├── Reduces sticking force
    └── May spot fluids
```

### Jarring Guidelines

| Phase | Overpull | Jar Blows | Duration |
|-------|----------|-----------|----------|
| Initial | Light (10-20k over) | 10-20 | 1-2 hours |
| Medium | Moderate (20-40k over) | 20-50 | 2-4 hours |
| Heavy | Maximum safe (40-60k+) | 50+ | 4-8 hours |
| Re-evaluate | - | - | After 8 hours |

### If Fish Won't Release

```
OPTIONS WHEN JARRING FAILS
├── Spotting Fluids
│   ├── Spot oil/acid around stuck point
│   ├── Soak time (4-24 hours)
│   ├── Resume jarring
│   └── May reduce differential
├── Washover
│   ├── Release fishing tool
│   ├── Run washover pipe
│   ├── Mill around fish
│   └── Re-engage and pull
├── Back-Off
│   ├── Back off above stuck point
│   ├── Retrieve free section
│   ├── Leave stuck portion
│   └── Sidetrack or mill
├── Milling
│   ├── Mill over fish top
│   ├── Destroy fish
│   ├── Recover in pieces
│   └── Last resort for tubulars
└── Abandonment
    ├── Set cement plug
    ├── Sidetrack around fish
    └── Document thoroughly
```

## Specific Fishing Scenarios

### Twist-Off Recovery

```
TWIST-OFF FISHING PROCEDURE
├── Assess Break
│   ├── Calculate twist-off depth (stretch)
│   ├── Examine recovered portion
│   ├── Determine fish top condition
│   └── Note connection orientation
├── Prepare for Engagement
│   ├── Dress fish top if needed
│   ├── Select appropriate tool
│   │   ├── Box up → Overshot or taper tap
│   │   └── Pin up → Spear or die collar
│   └── Consider pilot mill if damaged
├── Run Fishing Tool
│   ├── Tag fish carefully
│   ├── Engage fish
│   ├── Verify engagement
│   └── Attempt to pull
├── If Stuck
│   ├── Jar as needed
│   ├── Determine stuck point
│   ├── May need to back off higher
│   └── Consider washover
└── Post-Recovery
    ├── Inspect all recovered components
    ├── Check hole condition
    ├── May need to ream
    └── Continue operations
```

### Dropped Object Recovery

```
DROPPED OBJECT FISHING
├── Identify Object
│   ├── What was dropped?
│   ├── Dimensions and weight
│   ├── Material (magnetic?)
│   └── Where did it land?
├── Tool Selection
│   ├── Magnetic → Fishing magnet
│   ├── Small debris → Junk basket
│   ├── Larger object → Overshot/spear
│   └── Irregular shape → Reverse circulation
├── Fishing Procedure
│   ├── Run tool to depth
│   ├── Circulate while searching
│   ├── Tag bottom slowly
│   ├── Work tool over object
│   └── Retrieve
├── Multiple Passes
│   ├── Often requires several runs
│   ├── Check returns for debris
│   ├── May need to mill
│   └── Patience required
└── Verification
    ├── Confirm object recovered
    ├── Check hole clear
    ├── Run junk basket to clean
    └── Resume operations
```

### Wireline Tool Recovery

```
WIRELINE FISH RECOVERY
├── Determine Fish Type
│   ├── Logging tool string
│   ├── Perforating guns
│   ├── Plugs/packers
│   └── Wire/cable only
├── Special Considerations
│   ├── Small diameter fish
│   ├── Often in casing/tubing
│   ├── May have radioactive sources
│   └── Explosives (undetonated guns)
├── Tool Selection
│   ├── Small overshot
│   ├── Spear (if ID accessible)
│   ├── Magnet (for metal debris)
│   └── Specialized wireline fishing tools
├── Safety Considerations
│   ├── Radioactive sources → Radiation protocol
│   ├── Undetonated explosives → Extreme caution
│   ├── Wire under tension → Cut carefully
│   └── Fire perforating guns if possible before fishing
└── Recovery Procedure
    ├── Standard engagement
    ├── Gentle extraction
    ├── Account for all components
    └── Proper disposal of hazardous items
```

## Unsuccessful Fishing

### When to Stop Fishing

```
DECISION TO STOP FISHING
├── Exceeded Time Limit
│   ├── Pre-set limit reached
│   ├── No meaningful progress
│   └── Evaluate next steps
├── Exceeded Cost Limit
│   ├── Approaching sidetrack cost
│   ├── Diminishing returns
│   └── Management decision
├── Technical Barriers
│   ├── Cannot engage fish
│   ├── Fish damaged beyond recovery
│   ├── Hole conditions prevent access
│   └── Risk of making worse
└── Risk Assessment
    ├── Risk to string/tools
    ├── Wellbore damage risk
    ├── HSE considerations
    └── Stuck fishing tool risk
```

### Post-Fishing Operations

```
IF FISH NOT RECOVERED
├── Document Everything
│   ├── All attempts made
│   ├── Final fish position
│   ├── Tools used
│   └── Reason for stopping
├── Options
│   ├── Sidetrack (most common)
│   │   ├── Set cement plug
│   │   ├── Kick off above fish
│   │   └── Drill new section
│   ├── Mill Through
│   │   ├── Destroy fish with mills
│   │   ├── Time consuming
│   │   └── Use if fish blocking objective
│   └── Abandonment
│       ├── If well uneconomic
│       ├── Set cement plugs
│       └── P&A procedure
└── Lessons Learned
    ├── Root cause analysis
    ├── Prevention for future
    └── Share with team
```

## Safety Considerations

### Fishing Operation Hazards

| Hazard | Risk | Mitigation |
|--------|------|------------|
| Stuck fishing tool | High NPT, lost tool | Safety joints, limits |
| High tension failure | Personnel injury | Stand clear, limits |
| Jar misfire | Injury, equipment damage | Proper settings |
| String shot | Explosion hazard | Qualified personnel |
| Radioactive sources | Radiation exposure | NORM protocols |
| H2S exposure | Toxic exposure | Gas monitoring |

### Safety Protocols

```
FISHING SAFETY REQUIREMENTS
├── Personnel
│   ├── Experienced fishing hand
│   ├── Trained rig crew
│   └── Clear communication
├── Equipment
│   ├── Calibrated weight indicator
│   ├── Working deadline anchor
│   ├── Proper tongs and handling
│   └── Safety equipment available
├── Procedures
│   ├── Pre-job safety meeting
│   ├── Clear decision authority
│   ├── Defined limits
│   └── Emergency procedures
└── Monitoring
    ├── Continuous weight monitoring
    ├── Regular pressure checks
    ├── Communication maintained
    └── Document all activities
```
