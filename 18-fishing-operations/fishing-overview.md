### Fish Condition Assessment

| Condition | Description | Recovery Difficulty |
|-----------|-------------|-------------------|
| Free | Can be moved up/down | Easiest |
| Stuck (differential) | Held by pressure differential | Moderate |
| Stuck (mechanical) | Wedged, keyseat, packoff | Moderate-Difficult |
| Stuck (cement) | Set in cement | Very Difficult |
| Damaged | Crushed, deformed, broken | Difficult |
| Multiple pieces | Parted string, debris | Very Difficult |

## Fishing Decision Process

### Initial Assessment

```
IMMEDIATE ACTIONS AFTER FISH CREATED
├── Stop Operations
│   ├── Don't pump (may push fish deeper)
│   ├── Don't rotate (may damage fish top)
│   └── Document exact circumstances
├── Gather Information
│   ├── Confirm fish depth
│   ├── Calculate fish dimensions
│   ├── Review string tally
│   └── Note time fish created
├── Notify
│   ├── Company man
│   ├── Drilling engineer
│   ├── Management
│   └── Fishing company
└── Initial Assessment
    ├── What type of fish?
    ├── Is it stuck or free?
    ├── What tools are on rig?
    └── Preliminary recovery plan
```

### Economic Analysis

```
FISHING COST ESTIMATE
├── Direct Costs
│   ├── Fishing tools rental
│   ├── Fishing hands (day rate)
│   ├── Additional mud/materials
│   └── Support services
├── Rig Time Costs
│   ├── Daily rig rate × estimated days
│   ├── Spread cost (services, rentals)
│   └── Personnel costs
├── Potential Complications
│   ├── Stuck fishing tools (+days)
│   ├── Multiple fishing runs (+days)
│   └── Sidetrack if unsuccessful
└── Total Estimate
    └── Compare to sidetrack cost

SIDETRACK COST ESTIMATE
├── Cement plug
├── Whipstock or kickoff
├── Drilling new hole section
├── Additional casing/liner
├── Lost hole section work
└── Time to recover original trajectory

DECISION RULE
├── If Fishing Cost < 50% Sidetrack Cost: Fish first
├── If Fishing Cost ≈ Sidetrack Cost: Evaluate success probability
├── If Success Probability < 30%: Consider sidetrack
└── Always set maximum fishing time limit
```

### Success Probability Factors

| Factor | High Success | Low Success |
|--------|-------------|-------------|
| Time stuck | < 24 hours | > 72 hours |
| Hole condition | Good, stable | Sloughing, unstable |
| Fish top | Clean, accessible | Damaged, buried |
| Fish type | Standard tubular | Complex BHA, debris |
| Hole angle | < 30° | > 60° |
| Fish length | Short | Very long |
| Previous attempts | None | Multiple failed |

## Types of Stuck Pipe Leading to Fish

### Differential Sticking

```
DIFFERENTIAL STICKING
├── Mechanism
│   ├── Pressure differential pushes pipe against wall
│   ├── Filter cake acts as seal
│   ├── Common in permeable zones
│   └── More likely with OBM
├── Indicators
│   ├── Pipe stuck while static
│   ├── Cannot rotate or reciprocate
│   ├── Usually can circulate
│   └── Often at permeable formation
├── If Not Freed → Fish Characteristics
│   ├── Typically long fish
│   ├── Stuck section identifiable
│   └── May need to back off above stuck point
└── Prevention
    ├── Minimize stationary time
    ├── Optimize mud properties
    └── Use proper filter cake
```

### Mechanical Sticking

```
MECHANICAL STICKING TYPES
├── Keyseating
│   ├── Groove cut by tool joints
│   ├── Larger components hang up
│   ├── Common in deviated wells
│   └── Often leads to twist-off
├── Undergauge Hole
│   ├── Formation squeeze/heave
│   ├── Swelling shales
│   ├── Salt creep
│   └── Requires reaming
├── Junk in Hole
│   ├── Objects blocking passage
│   ├── May prevent rotation
│   └── Often leads to twist-off
├── Packoff
│   ├── Cuttings accumulation
│   ├── Caving/sloughing
│   ├── Bridges around string
│   └── Usually gradual
└── Cement
    ├── Set around pipe
    ├── Float failure
    └── Very difficult to free
```

### Twist-Off Characteristics

```
TWIST-OFF ANALYSIS
├── Location Indicators
│   ├── Calculate stretch to verify depth
│   ├── Review torque/drag history
│   ├── Check for previous damage
│   └── Note washout indications
├── Common Twist-Off Points
│   ├── Worn connection
│   ├── Slip damage area
│   ├── Fatigue crack location
│   ├── Corrosion pit
│   └── Previous repair
├── Fish Top Condition
│   ├── Likely damaged/ragged
│   ├── May be mushroomed
│   ├── Threading damaged
│   └── May need to dress before engaging
└── Documentation
    ├── Record exact depth
    ├── Photograph retrieved portion
    ├── Note failure mode
    └── Save sample for analysis
```

## Fishing Planning

### Pre-Fishing Checklist

```
BEFORE FISHING OPERATIONS
├── Fish Definition
│   ├── Type and dimensions confirmed
│   ├── Top of fish depth verified
│   ├── Fish condition assessed
│   └── Stuck or free determined
├── Wellbore Information
│   ├── Hole size and condition
│   ├── Casing setting depths
│   ├── Mud type and weight
│   ├── Temperature
│   └── H2S presence
├── Tool Selection
│   ├── Primary fishing tool
│   ├── Backup tools
│   ├── Accessories (jar, bumper sub)
│   └── Milling tools if needed
├── Personnel
│   ├── Fishing hand on location
│   ├── Tool company support
│   └── Decision maker identified
└── Limits Established
    ├── Maximum time
    ├── Maximum cost
    ├── Decision points
    └── Sidetrack trigger
```

### Fishing BHA Design

```
TYPICAL FISHING BHA (Top to Bottom)
├── Drill Pipe
├── Safety Joint
│   └── Allows release if fishing tool stuck
├── Jar (Hydraulic or Mechanical)
│   └── Provides impact force
├── Bumper Sub / Accelerator
│   └── Increases jar effectiveness
├── Drill Collars (for weight)
├── Safety Joint (optional second)
├── Fishing Tool
│   ├── Overshot, or
│   ├── Spear, or
│   ├── Mill, etc.
└── Accessories
    ├── Washover pipe (if needed)
    ├── Guide (if needed)
    └── Magnets (for junk)

WEIGHT CONSIDERATIONS
├── Enough weight to set jar
├── Not so much to buckle string
├── Consider hole angle
└── Match to fishing tool requirements
```

### Time and Cost Limits

| Phase | Recommended Limit | Action if Exceeded |
|-------|------------------|-------------------|
| Initial engagement | 8-12 hours | Evaluate, try alternative |
| Freeing operations | 24-48 hours | Management review |
| Total fishing | 3-5 days | Sidetrack decision |
| Cost | 50-75% of sidetrack | Serious evaluation |
