# Severe Weather

## Overview

Severe weather can create dangerous conditions on drilling locations. The Company Man is responsible for monitoring weather conditions and implementing appropriate precautions to protect personnel and equipment.

---

### Information Sources

| Source | Use |
|--------|-----|
| **National Weather Service** | Official forecasts, warnings |
| **Commercial services** | Detailed forecasts |
| **Weather radio** | Real-time alerts |
| **Internet/apps** | Radar, forecasts |

| Condition | Monitoring |
|-----------|------------|
| **Normal conditions** | Twice daily |
| **Unsettled weather** | Every 2-4 hours |

## Lightning

### Lightning Safety

**30/30 Rule**:
- If thunder follows lightning by 30 seconds or less, seek shelter
- Wait 30 minutes after last lightning/thunder before resuming

### When Lightning Threatens

| Action | When |
|--------|------|
| **Monitor** | Storm approaching |
| **Suspend high-risk work** | Thunder heard |
| **Seek shelter** | Lightning within 6 miles |
| **Evacuate rig floor** | Storm overhead |
| **Resume** | 30 minutes after last activity |

### Lightning Response

```

│                    LIGHTNING PROTOCOL                           │

│                                                                 │
│   LIGHTNING WITHIN 3 MILES                                      │
│   ├── All personnel seek shelter                                │
│   ├── Stay in enclosed structures or vehicles                   │
│   └── Avoid water, trees, open areas                            │
│                                                                 │

```

### If Someone is Struck

- It is safe to touch a lightning strike victim
- Call 911 immediately
- Begin CPR if not breathing
- Treat for burns and shock

## Tornadoes

### Tornado Watch vs. Warning

| Alert | Meaning | Action |
|-------|---------|--------|
| **Watch** | Conditions favorable | Monitor, prepare |
| **Warning** | Tornado sighted/indicated | Take shelter immediately |

### Tornado Precautions

**When Watch is Issued**:
- Monitor weather continuously
- Review shelter locations
- Prepare to secure equipment
- Keep personnel informed

**When Warning is Issued**:
- Take shelter immediately
- Move to lowest level
- Get away from windows
- Cover and protect head

### Shelter Locations

| Best | Worst |
|------|-------|
| Storm cellar | Vehicles |
| Interior room, lowest floor | Mobile trailers |
| Under sturdy furniture | Near windows |

### If No Shelter Available

- Lie flat in nearest ditch or low area
- Cover head with arms
- Stay away from vehicles and trees

## Hurricanes

### Hurricane Categories

| Category | Wind Speed | Damage |
|----------|------------|--------|
| 1 | 74-95 mph | Minimal |
| 2 | 96-110 mph | Moderate |
| 3 | 111-129 mph | Extensive |
| 4 | 130-156 mph | Extreme |
| 5 | >156 mph | Catastrophic |

### Hurricane Preparation

**72 Hours Before**:
- Monitor forecast tracks
- Review evacuation plans
- Inventory emergency supplies

**48 Hours Before**:
- Secure loose equipment
- Fill fuel and water tanks
- Prepare to evacuate

**24 Hours Before**:
- Final securing of equipment
- Decision to evacuate
- Execute evacuation if ordered

### Evacuation Decision

| Factor | Consider |
|--------|----------|
| **Storm track** | Projected path |
| **Intensity** | Category forecast |
| **Timing** | When will impact occur |
| **Personnel** | Time to evacuate |
| **Equipment** | Securing needs |

## Winter Weather

### Winter Storm Precautions

| Condition | Action |
|-----------|--------|
| **Watch issued** | Prepare supplies, monitor |
| **Warning issued** | Implement cold weather procedures |
| **Blizzard** | Shelter in place if caught |

### Cold Weather Operations

| Temperature | Precautions |
|-------------|-------------|
| **Below 40°F** | Cold weather gear available |
| **Below 32°F** | Freeze protection active |
| **Below 0°F** | Limited outdoor work, warm-up breaks |

### Freeze Protection

| System | Protection |
|--------|------------|
| **Water lines** | Heat trace, insulation, drain |
| **Mud system** | Heaters, circulation |
| **BOP** | Heat, circulation |
| **Fuel** | Winter blend, treatment |
| **Personnel** | Cold weather PPE, warming stations |

### Ice Hazards

| Hazard | Mitigation |
|--------|------------|
| **Ice on walkways** | Salt/sand, anti-icing |
| **Ice on stairs** | Handrails, treatment |
| **Ice on equipment** | De-icing, heating |
| **Icicles** | Removal, barriers |

## Extreme Heat

### Heat Advisory Levels

| Condition | Heat Index | Action |
|-----------|------------|--------|
| **Caution** | 80-89°F | Fatigue possible |
| **Extreme Caution** | 90-102°F | Heat illness possible |
| **Danger** | 103-124°F | Heat illness likely |
| **Extreme Danger** | >125°F | Heat stroke imminent |

### Hot Weather Precautions

| Precaution | Implementation |
|------------|----------------|
| **Hydration** | Water available everywhere |
| **Rest cycles** | More frequent breaks |
| **Shade** | Provide shade structures |
| **Scheduling** | Avoid hottest hours |
| **Monitoring** | Watch for heat illness |
| **Acclimatization** | Gradual exposure for new workers |

## High Winds

### Wind Speed Actions

| Wind Speed | Actions |
|------------|---------|
| **25-30 mph** | Caution with crane ops |
| **30-35 mph** | Suspend crane operations |
| **35-40 mph** | Secure loose items |
| **>40 mph** | Suspend elevated work |
| **>50 mph** | Consider evacuation |

### Wind Precautions

| Action | When |
|--------|------|
| **Secure loose items** | Winds increasing |
| **Lower/secure equipment** | High winds forecast |
| **Monitor conditions** | Continuously |
| **Suspend crane work** | Per manufacturer limits |
| **Suspend work at height** | Winds >40 mph |

## Fog

### Fog Precautions

| Visibility | Action |
|------------|--------|
| **< 1/4 mile** | Reduce travel |
| **< 500 ft** | Suspend lifting operations |
| **< 100 ft** | Only essential travel |
| **Near zero** | Suspend vehicle movement |

### Fog Response

- Increase lighting
- Use spotters for operations
- Slow vehicle speeds
- Delay helicopter operations
- Use audible signals

## Flooding

### Flash Flood Response

**Watch**: Conditions favorable for flash flooding
**Warning**: Flash flooding occurring or imminent

### Flood Precautions

| Action | Purpose |
|--------|---------|
| **Monitor levels** | Track water rise |
| **Move equipment** | Away from flood zones |
| **Evacuate** | If flooding threatens |
| **Never drive through** | Turn around, don't drown |

### After Flooding

| Action | Purpose |
|--------|---------|
| **Assess damage** | Before re-entry |
| **Check utilities** | Electrical hazards |
| **Document** | For insurance/reporting |
| **Clean up** | Safely, with PPE |
