# Security Threats

## Overview

Security threats at drilling locations can include theft, vandalism, unauthorized access, and personal safety threats. The Company Man is responsible for maintaining a secure location and knowing how to respond to security incidents.

---

## Site Security

### Access Control

| Control | Purpose |
|---------|---------|
| **Controlled entry** | Single point of access |
| **Sign-in log** | Track all personnel |
| **Visitor escort** | Accompanied at all times |
| **ID verification** | Confirm identity |
| **Vehicle registry** | Track vehicles on site |

### Perimeter Security

| Measure | Application |
|---------|-------------|
| **Fencing** | Where practical |
| **Gates** | Locked when unmanned |
| **Lighting** | Adequate for visibility |
| **Signage** | No trespassing, authorized only |
| **Cameras** | If available |

### Key Control

| Practice | Purpose |
|----------|---------|
| **Limited distribution** | Only those who need |
| **Sign-out log** | Track key custody |
| **Regular inventory** | Account for all keys |
| **Re-key** | When personnel leave |

## Theft Prevention

### High-Value Items

| Item | Protection |
|------|------------|
| **Equipment** | Inventory, GPS trackers |
| **Tools** | Lock up, tool room control |
| **Fuel** | Secured tanks, monitors |
| **Electronics** | Secure storage, inventories |
| **Vehicles** | Keys secured, GPS |

### Theft Deterrence

| Measure | Action |
|---------|--------|
| **Lighting** | Illuminate vulnerable areas |
| **Visibility** | Reduce hiding places |
| **Presence** | Security personnel if needed |
| **Inventory** | Regular checks |
| **Reporting** | Encourage reporting of suspicious activity |

### If Theft Occurs

| Step | Action |
|------|--------|
| 1 | Report to supervisor |
| 2 | Document what was taken |
| 3 | Do not disturb scene |
| 4 | Notify law enforcement |
| 5 | File police report |
| 6 | Report to company |

## Suspicious Activity

### What to Look For

| Activity | Possible Concern |
|----------|------------------|
| **Unknown vehicles** | Surveillance, planning |
| **Photographing** | Intelligence gathering |
| **Unusual questions** | Social engineering |
| **Loitering** | Opportunity seeking |
| **Repeated visits** | Pattern development |

### Response to Suspicious Activity

| Action | Details |
|--------|---------|
| **Observe** | Note details (description, vehicle, license) |
| **Don't confront** | Safety first |
| **Report** | To supervisor, security |
| **Document** | Written record |
| **Follow up** | Track patterns |

|---------|----------------|

| **Reporting** | Easy, confidential |
| **Training** | Recognize, report |

| **Response plan** | For incidents |

### De-Escalation

| Technique | Application |
|-----------|-------------|
| **Stay calm** | Don't match aggression |
| **Listen** | Let them talk |
| **Acknowledge** | Validate concerns |
| **Space** | Maintain safe distance |
| **Avoid confrontation** | Don't argue or challenge |
| **Exit strategy** | Know your escape |

## Active Threat Response

### RUN - HIDE - FIGHT

```
┌─────────────────────────────────────────────────────────────────┐
│                  ACTIVE THREAT RESPONSE                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   RUN (First option)                                            │
│   ├── Leave immediately if safe path exists                    │
│   ├── Leave belongings behind                                   │
│   ├── Help others escape if possible                           │
│   ├── Keep hands visible                                        │
│   └── Call 911 when safe                                       │
│                                                                 │
│   HIDE (If running not possible)                               │
│   ├── Find a secure location                                    │
│   ├── Lock and barricade door                                   │
│   ├── Turn off lights                                           │
│   ├── Silence phone                                             │
│   ├── Stay quiet                                                │
│   └── Call 911 if safe to do so                                │
│                                                                 │
│   FIGHT (Last resort only)                                      │
│   ├── Commit fully                                              │
│   ├── Use improvised weapons                                    │
│   ├── Act with aggression                                       │
│   └── Incapacitate the threat                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### When Law Enforcement Arrives

| Action | Purpose |
|--------|---------|
| **Drop anything in hands** | Avoid appearing as threat |
| **Raise hands, spread fingers** | Show you're not armed |
| **Follow commands** | Exactly as given |
| **Stay calm** | Don't make sudden movements |
| **Don't point or yell** | Let officers assess |

## Bomb Threat

### If Threat Received

| Action | Details |
|--------|---------|
| **Stay calm** | Keep caller talking |
| **Listen carefully** | Note exact words |
| **Ask questions** | Where, when, what, why, who |
| **Note details** | Voice, background, caller ID |
| **Don't hang up** | If possible |
| **Alert others** | Signal for help |

### Bomb Threat Questions

| Question | Information |
|----------|-------------|
| When will it explode? | Time |
| Where is it? | Location |
| What does it look like? | Description |
| What will cause it to explode? | Trigger |
| Did you place it? | Responsibility |
| Why? | Motive |
| Who are you? | Identity |

### Evacuation Decision

| Factor | Consider |
|--------|----------|
| **Credibility** | How specific is threat? |
| **History** | Previous threats? |
| **Current events** | Heightened risks? |
| **Time** | Enough to evacuate? |
| **Authority** | Law enforcement guidance |

### If Suspicious Package Found

| Action | Details |
|--------|---------|
| **Do NOT touch** | Leave it alone |
| **Do NOT use radio** | Could trigger |
| **Evacuate area** | Clear the zone |
| **Call authorities** | From safe distance |
| **Describe** | Location, appearance |

## Civil Unrest

### Precautions

| Action | Purpose |
|--------|---------|
| **Monitor news** | Awareness of situations |
| **Avoid flashpoints** | Stay away from gatherings |
| **Secure location** | Limit access |
| **Maintain low profile** | Don't attract attention |
| **Prepare to evacuate** | If situation escalates |

### If Encountered

| Action | Details |
|--------|---------|
| **Leave immediately** | If safe to do so |
| **Don't engage** | Not worth the risk |
| **Stay in vehicle** | Doors locked |
| **Seek alternate routes** | Avoid crowds |
| **Report** | To authorities and company |

## Reporting Security Incidents

### What to Report

| Incident | Report |
|----------|--------|
| **Any theft** | Regardless of value |
| **Suspicious activity** | All observations |
| **Threats** | Verbal or written |
| **Vandalism** | Any damage |
| **Trespassing** | Unauthorized entry |
| **Violence** | Any incident |

### How to Report

| Report To | When |
|-----------|------|
| **Immediate supervisor** | Immediately |
| **Law enforcement** | Crimes |
| **Company security** | All incidents |

### Information to Provide

| Information | Details |
|-------------|---------|
| **What happened** | Description of incident |
| **When** | Date and time |
| **Where** | Location |
| **Who** | Persons involved, witnesses |
| **Description** | Physical, vehicle, etc. |
