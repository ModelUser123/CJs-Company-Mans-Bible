# Fire & Explosion Response

## Overview

Fires and explosions on drilling and production locations can occur quickly and escalate rapidly. Quick recognition, proper response, and well-trained personnel are essential for minimizing injuries and damage.

---

## Fire Prevention

### Common Ignition Sources

| Source | Mitigation |
|--------|------------|
| **Hot work** | Permits, fire watch, gas testing |
| **Electrical** | Proper classification, maintenance |
| **Static** | Grounding, bonding |
| **Friction** | Maintenance, lubrication |
| **Hot surfaces** | Guards, insulation |
| **Open flame** | Elimination, permits |
| **Smoking** | Designated areas only |

### Fuel Sources

| Fuel | Location |
|------|----------|
| **Crude oil** | Flow lines, tanks |
| **Natural gas** | Well, process |
| **Diesel** | Fuel storage, engines |
| **Propane** | Heating, equipment |
| **Drilling fluids** | OBM, SBM |
| **Chemicals** | Various locations |

## Fire Classification

### Fire Classes

| Class | Fuel Type | Examples |
|-------|-----------|----------|
| **A** | Ordinary combustibles | Wood, paper, cloth |
| **B** | Flammable liquids/gases | Oil, diesel, gas |
| **C** | Electrical | Energized equipment |
| **D** | Combustible metals | Magnesium, sodium |
| **K** | Cooking oils | Kitchen fires |

### Extinguisher Selection

| Extinguisher Type | Effective On |
|-------------------|--------------|
| **ABC (dry chemical)** | A, B, C |
| **BC (dry chemical)** | B, C |
| **CO2** | B, C |
| **Water** | A only (NOT on oil/electrical) |
| **Foam** | A, B |

|------|--------|

## When to Fight vs. Evacuate

### Fight the Fire If:

| Condition | Status |
|-----------|--------|
| Fire is small | Trash can size or smaller |
| You are trained | Know how to use extinguisher |
| Correct extinguisher | Right type for fire class |
| Exit is clear | Escape route behind you |
| Others are alerted | Alarm sounded |
| Area is ventilated | Smoke not excessive |

### Evacuate If:

| Condition | Action |
|-----------|--------|
| Fire is spreading | Do NOT attempt |
| Smoke is heavy | Cannot see |
| Wrong extinguisher | Wrong type or empty |
| Exit is blocked | Fire between you and exit |
| Unsure of contents | Unknown hazards |
| Instinct says go | Trust your instincts |

## Specific Fire Scenarios

### Wellhead Fire

| Action | Details |
|--------|---------|
| **Evacuate** | All personnel to safe distance |
| **Shut in** | If possible, close valves remotely |
| **Let it burn** | May be safer than unignited |
| **Call experts** | Well control company |
| **Do NOT approach** | Heat and pressure hazards |

### Engine/Equipment Fire

| Action | Details |
|--------|---------|
| **Shut down** | Stop engine if possible |
| **Isolate fuel** | Close valves, shut pumps |
| **Extinguish** | ABC or BC extinguisher |
| **Electrical** | De-energize before water |

### Tank Fire

| Action | Details |
|--------|---------|
| **Evacuate** | Potential BLEVE risk |
| **Call fire department** | Do NOT fight alone |
| **Cool adjacent tanks** | Prevent spread |
| **Monitor** | For boilover or explosion |

### Electrical Fire

| Action | Details |
|--------|---------|
| **De-energize** | If safe to do so |
| **CO2 or BC extinguisher** | NEVER water |
| **Call electrician** | After fire out |

## Fire Watch

### When Required

| Situation | Fire Watch Duration |
|-----------|---------------------|
| **Hot work** | During + 30 minutes after |
| **Welding near combustibles** | Continuous |
| **Cutting with torch** | Continuous |
| **Grinding** | During work |

### Fire Watch Duties

| Duty | Action |
|------|--------|
| **Observe** | Watch for ignition |
| **Equipped** | Fire extinguisher ready |
| **Trained** | Know how to use extinguisher |
| **Alert** | Sound alarm if fire starts |
| **Stay** | Remain for full duration |
| **No other duties** | Fire watch only |

## Explosion Hazards

### Explosion Prevention

| Hazard | Control |
|--------|---------|
| **Gas accumulation** | Ventilation, monitoring |
| **Dust** | Housekeeping, suppression |
| **Confined space** | Purging, testing |
| **Chemical reaction** | Separation, procedures |

### Explosion Response

| Action | Details |
|--------|---------|
| **Take cover** | Protect from secondary effects |
| **Evacuate** | Move away from area |
| **Account for personnel** | Headcount |
| **Do NOT re-enter** | Until declared safe |
| **Call for help** | Emergency services |

## After the Fire

### Immediate Actions

| Action | Purpose |
|--------|---------|
| **Overhaul** | Ensure fire is out |
| **Cool hot spots** | Prevent reignition |
| **Ventilate** | Remove smoke |
| **Assess damage** | Document |
| **Investigate** | Determine cause |

### Investigation

| Element | Focus |
|---------|-------|
| **Origin** | Where did it start? |
| **Cause** | What started it? |
| **Fuel** | What burned? |
| **Ignition** | Source of ignition? |
| **Response** | How was response? |
| **Improvements** | What can we do better? |

---

## Fire Response Checklist

### Fire Discovered
- [ ] Sound alarm
- [ ] Call emergency services
- [ ] Rescue anyone in immediate danger
- [ ] Assess fire (size, type)
- [ ] Fight if small and safe
- [ ] Evacuate if large or spreading
- [ ] Account for all personnel

### After Fire
- [ ] Ensure fire is completely out
- [ ] Ventilate area
- [ ] Document damage
- [ ] Investigate cause
- [ ] Report to company
- [ ] Implement corrective actions
