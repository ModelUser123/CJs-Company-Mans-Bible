## Key Concepts

### Emergency Response Priority

```
┌─────────────────────────────────────────────────┐
│         EMERGENCY RESPONSE PRIORITIES           │
├─────────────────────────────────────────────────┤
│                                                 │
│   1. PROTECT LIFE     → Evacuate, rescue       │
│          ↓                                      │
│   2. PROTECT ENVIRONMENT → Contain, mitigate   │
│          ↓                                      │
│   3. PROTECT ASSETS   → Secure equipment       │
│          ↓                                      │
│   4. RESTORE OPERATIONS → Resume safely        │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Emergency Types and Response

| Emergency | Initial Response | Key Actions |
|-----------|-----------------|-------------|
| **Well Control** | Shut in, evacuate danger zone | Implement kill procedures |
| **H2S Release** | Evacuate upwind, account for all | Rescue with SCBA only |
| **Fire** | Alarm, evacuate, fight if safe | Shut in fuel sources |
| **Medical** | First aid, call for help | MEDEVAC if needed |
| **Severe Weather** | Shelter/evacuate per plan | Secure equipment |
| **Security Threat** | Notify authorities, secure | Follow company protocol |

### Critical Success Factors

1. **Preparation** - Plan before the emergency
2. **Training** - Regular drills for all scenarios
3. **Communication** - Clear chains of command
4. **Equipment** - Maintained and accessible
5. **Documentation** - Record everything during and after
