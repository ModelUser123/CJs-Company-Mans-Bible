# Medical Emergency

## Overview

Medical emergencies on remote drilling locations require immediate action and coordination with emergency services. The Company Man ensures medical emergency plans are in place and personnel are trained.

---

## First Aid Basics

### First Aid Priorities

```
D - DANGER     ─── Is the scene safe for you?
R - RESPONSE   ─── Is the person responsive?
S - SEND       ─── Send for help (call 911/MEDEVAC)
A - AIRWAY     ─── Is the airway clear?
B - BREATHING  ─── Is the person breathing?
C - CPR        ─── Start CPR if not breathing
D - DEFIBRILLATION ─ Use AED if available
```

### When to Call Emergency Services

**Always call for**:
- Unconsciousness
- Difficulty breathing
- Chest pain/pressure
- Severe bleeding
- Severe burns
- Head/neck/spine injury
- Suspected stroke
- Severe allergic reaction
- Amputation
- Seizures

### Universal Precautions

| Precaution | Purpose |
|------------|---------|
| **Gloves** | Protect from blood/bodily fluids |
| **Face shield** | If splash risk |
| **Hand washing** | After any contact |
| **Proper disposal** | Contaminated materials |

## CPR Procedures

### Adult CPR

| Step | Action |
|------|--------|
| **1** | Check scene safety |
| **2** | Check responsiveness (tap, shout) |
| **3** | Call 911, get AED |
| **4** | Check breathing (10 seconds max) |
| **5** | If not breathing, start CPR |
| **6** | 30 compressions, 2 breaths |
| **7** | Continue until help arrives |

### Compression Guidelines

| Parameter | Adult |
|-----------|-------|
| **Hand position** | Center of chest |
| **Depth** | At least 2 inches |
| **Rate** | 100-120 per minute |
| **Ratio** | 30:2 (compressions:breaths) |
| **Allow recoil** | Full chest recoil between |

### AED Use

| Step | Action |
|------|--------|
| **1** | Turn on AED |
| **2** | Bare chest, dry if wet |
| **3** | Apply pads as shown |
| **4** | Stand clear during analysis |
| **5** | Deliver shock if advised |
| **6** | Resume CPR immediately |

## Common Oilfield Injuries

### Trauma

| Injury | First Aid |
|--------|-----------|
| **Cuts** | Direct pressure, elevate, bandage |
| **Amputations** | Control bleeding, save part in ice |
| **Fractures** | Immobilize, don't move spine injuries |
| **Crush injuries** | Call for help, monitor for shock |

### Burns

| Degree | Treatment |
|--------|-----------|
| **First (red)** | Cool water, cover loosely |
| **Second (blisters)** | Cool water, don't pop blisters |
| **Third (charred)** | Cover, treat for shock, MEDEVAC |
| **Chemical** | Flush 20+ minutes, remove clothing |

### Eye Injuries

| Type | Treatment |
|------|-----------|
| **Chemical splash** | Flush 15-20 minutes |
| **Foreign object** | Flush, don't rub |
| **Impact** | Cover both eyes, seek help |
| **Arc flash** | Cover, cool compress, seek help |

## Heat-Related Illness

### Heat Exhaustion

| Symptoms | Treatment |
|----------|-----------|
| Heavy sweating | Move to cool area |
| Pale, clammy skin | Remove excess clothing |
| Nausea | Cool water to drink |
| Weakness | Apply cool, wet cloths |
| Rapid pulse | Monitor closely |

### Heat Stroke - MEDICAL EMERGENCY

| Symptoms | Treatment |
|----------|-----------|
| High body temp (>103°F) | Call 911 immediately |
| Red, hot, dry skin | Move to cool area |
| No sweating | Immerse in cold water if possible |
| Rapid, strong pulse | Apply ice packs to neck, armpits, groin |
| Confusion | Do NOT give fluids if altered |
| Unconsciousness | Be ready for CPR |

### Heat Illness Prevention

| Prevention | Action |
|------------|--------|
| **Hydration** | Drink before feeling thirsty |
| **Rest** | Take breaks in shade |
| **Acclimatization** | Adjust to heat gradually |
| **Clothing** | Light colored, loose fitting |
| **Buddy system** | Watch each other |

---

## Cold-Related Illness

### Hypothermia

| Severity | Symptoms | Treatment |
|----------|----------|-----------|
| **Mild** | Shivering, fumbling | Warm drinks, dry clothes, blankets |
| **Moderate** | Violent shivering, slurred speech | Remove wet clothes, warm slowly |
| **Severe** | Shivering stops, stiff muscles | Call 911, handle gently, warm slowly |

### Frostbite

| Stage | Treatment |
|-------|-----------|
| **Frostnip** | Warm gradually, don't rub |
| **Superficial** | Warm in tepid water (not hot) |
| **Deep** | Don't thaw if refreezing possible, MEDEVAC |

## MEDEVAC Procedures

### When to Request MEDEVAC

| Condition | MEDEVAC |
|-----------|---------|
| **Immediately life-threatening** | Yes |
| **Time-critical injury** | Yes |
| **Remote location** | Often needed |
| **Ground transport too slow** | Yes |

### MEDEVAC Information Needed

| Information | Details |
|-------------|---------|
| **Location** | Coordinates, landmarks |
| **Patient condition** | Injuries, vital signs |
| **Number of patients** | How many |
| **Mechanism of injury** | What happened |
| **Weather conditions** | Visibility, wind |
| **Hazards** | Power lines, gas, etc. |
| **Landing zone** | Size, surface, obstacles |

### Landing Zone Preparation

| Requirement | Details |
|-------------|---------|
| **Size** | Minimum 100 ft × 100 ft |
| **Surface** | Level, firm |
| **Obstacles** | None (wires, poles, equipment) |
| **Marking** | Lights or markers at corners |
| **Security** | Clear of people and vehicles |
| **Wind direction** | Indicator for pilot |

## Bloodborne Pathogen Exposure

### If Exposed

| Exposure | Action |
|----------|--------|
| **Needlestick** | Wash with soap and water |
| **Splash to mucous membrane** | Flush with water |
| **Splash to non-intact skin** | Wash with soap and water |

### Post-Exposure

| Step | Action |
|------|--------|
| 1 | Report to supervisor immediately |
| 2 | Document incident |
| 3 | Seek medical evaluation |
| 4 | Follow up per medical advice |
