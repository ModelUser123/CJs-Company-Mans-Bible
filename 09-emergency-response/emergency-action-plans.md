# Emergency Action Plans

## Overview

An Emergency Action Plan (EAP) provides the framework for responding to emergencies. Every person on location must know the plan, their role, and how to respond. The Company Man is responsible for ensuring the plan is in place, current, and practiced.

---

## Emergency Action Plan Requirements

### Required Elements

| Element | Content |
|---------|---------|
| **Emergency contacts** | Internal and external numbers |
| **Evacuation procedures** | Routes, assembly points |
| **Alarm systems** | How to recognize each alarm |
| **Roles and responsibilities** | Who does what |
| **Site map** | Evacuation routes, equipment locations |
| **Communication plan** | How to communicate during emergency |
| **Training requirements** | What training is needed |
| **Drill schedule** | When drills are conducted |

### Emergency Types Covered

| Emergency | Plan Required |
|-----------|---------------|
| **Fire/explosion** | Yes |
| **Well control** | Yes (see Well Control section) |
| **H2S release** | Yes (if applicable) |
| **Medical emergency** | Yes |
| **Severe weather** | Yes |
| **Security threat** | Yes |
| **Environmental release** | Yes |
| **Utility failure** | Yes |

## Alarm Systems

### Alarm Types

| Alarm | Signal | Meaning |
|-------|--------|---------|
| **General alarm** | Continuous horn | Emergency, prepare to evacuate |
| **Evacuation** | Intermittent horn | Evacuate immediately |
| **H2S alarm** | Distinct tone + strobe | H2S detected, don breathing apparatus |
| **Fire alarm** | Bell or siren | Fire detected |
| **All clear** | Steady tone or PA | Emergency over |

### Alarm Response

```
┌─────────────────────────────────────────────────────────────────┐
│                      ALARM RESPONSE                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   GENERAL ALARM (Continuous)                                    │
│   ├── Stop work                                                 │
│   ├── Assess situation                                          │
│   ├── Prepare to evacuate                                       │
│   └── Await further instructions                                │
│                                                                 │
│   EVACUATION ALARM (Intermittent)                              │
│   ├── Stop work immediately                                     │
│   ├── Secure immediate area if safe                            │
│   ├── Proceed to muster point                                   │
│   └── Report for headcount                                      │
│                                                                 │
│   H2S ALARM (Distinct tone + strobe)                           │
│   ├── Don breathing apparatus immediately                       │
│   ├── Move upwind to safe briefing area                        │
│   ├── Report for headcount                                      │
│   └── Do NOT attempt rescue without SCBA                       │
│                                                                 │
│   ALL CLEAR                                                     │
│   ├── Emergency is over                                         │
│   ├── Return to normal operations                               │
│   └── Participate in debrief if requested                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Evacuation Procedures

### Evacuation Routes

| Requirement | Details |
|-------------|---------|
| **Primary route** | Main exit from each area |
| **Secondary route** | Alternate if primary blocked |
| **Marking** | Clearly marked, illuminated |
| **Clear** | Unobstructed at all times |
| **Posted** | Map at key locations |

### Muster Points (Assembly Areas)

| Requirement | Details |
|-------------|---------|
| **Location** | Safe distance, upwind |
| **Marking** | Clearly identified |
| **Capacity** | Accommodate all personnel |
| **Accessibility** | Easy access from all areas |
| **Alternates** | Backup if primary unusable |

### Personnel Accountability

| Step | Action |
|------|--------|
| 1 | All personnel proceed to muster point |
| 2 | Report to designated person |
| 3 | Headcount conducted |
| 4 | Missing persons identified |
| 5 | Report to Incident Commander |
| 6 | Initiate search if safe to do so |

### Headcount Procedure

```
┌─────────────────────────────────────────────────────────────────┐
│                     HEADCOUNT PROCEDURE                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. Each crew supervisor counts their personnel                │
│                                                                 │
│   2. Visitors/contractors account for themselves                │
│                                                                 │
│   3. Reports to Safety Officer:                                 │
│      "Drill crew: 5 present, all accounted for"                │
│      OR                                                         │
│      "Mud crew: 3 present, 1 MISSING - John Smith"             │
│                                                                 │
│   4. Safety Officer compiles and reports to IC                 │
│                                                                 │
│   5. If anyone missing:                                         │
│      - Last known location?                                     │
│      - Search if safe                                           │
│      - Notify emergency services                                │
│                                                                 │
│   TIME TARGET: Complete headcount within 5 minutes             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## Emergency Drills

### Drill Requirements

| Drill Type | Frequency |
|------------|-----------|
| **Fire/evacuation** | Quarterly |
| **H2S (if applicable)** | Quarterly |
| **Man overboard (offshore)** | Monthly |
| **Abandon platform (offshore)** | Semi-annually |
| **Well control** | Per well control requirements |
| **SCBA donning** | Monthly |

### Drill Evaluation

| Criterion | Standard |
|-----------|----------|
| **Alarm recognition** | All personnel respond |
| **Evacuation time** | < 5 minutes to muster |
| **Headcount** | 100% accounted |
| **Equipment use** | Correct and timely |
| **Communication** | Clear and effective |

### Drill Documentation

| Record | Content |
|--------|---------|
| **Date and time** | When drill conducted |
| **Type of drill** | What was practiced |
| **Participants** | Who participated |
| **Observations** | What went well/poorly |
| **Corrective actions** | Improvements needed |
| **Follow-up** | Actions completed |
