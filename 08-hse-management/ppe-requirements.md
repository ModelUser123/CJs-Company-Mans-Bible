# PPE Requirements

## Overview

Personal Protective Equipment (PPE) is the last line of defense against workplace hazards. Proper selection, use, and maintenance of PPE is critical for worker safety.

---

## Minimum PPE Requirements

### Standard Site PPE

| PPE Item | Requirement |
|----------|-------------|
| **Hard hat** | ANSI Z89.1, Type I or II |
| **Safety glasses** | ANSI Z87.1 rated |
| **Steel-toe boots** | ASTM F2413, 6" or 8" |
| **FRC clothing** | NFPA 2112, 8 cal/cmÂ² minimum |
| **Hearing protection** | NRR 25+ in high noise |
| **Gloves** | Task-appropriate |

## Head Protection

### Hard Hat Requirements

| Feature | Requirement |
|---------|-------------|
| **Standard** | ANSI Z89.1 |
| **Type** | Type I (top) or Type II (top + side) |
| **Class** | E (Electrical), G (General), or C |
| **Suspension** | 4 or 6 point |

### Hard Hat Care

| Action | Frequency |
|--------|-----------|
| **Inspect** | Before each use |
| **Clean** | As needed with mild soap |
| **Replace** | If cracked, dented, UV damaged |
| **Suspension** | Replace annually |
| **Shell** | Replace per manufacturer (typically 5 years) |

### Prohibited Modifications

- No holes drilled
- No paint or stickers covering damage
- No backward wear (unless designed for it)
- No alterations to suspension

## Eye and Face Protection

### Safety Glasses

| Standard | ANSI Z87.1 |
|----------|-----------|
| **Marking** | Z87+ on frame and lens |
| **Side shields** | Required |
| **Tint** | Appropriate for conditions |

### Face Shields

| Use | When Required |
|-----|---------------|
| **Chemical handling** | Splashes possible |
| **Grinding** | In addition to glasses |
| **Pressure washing** | High-pressure sprays |
| **Hot work** | Correct shade for operation |

### Welding Lens Shades

| Operation | Shade Number |
|-----------|--------------|
| **Gas welding (light)** | 4-5 |
| **Gas welding (heavy)** | 5-6 |
| **Arc welding (up to 60A)** | 7-8 |
| **Arc welding (60-160A)** | 10-11 |
| **Arc welding (160-250A)** | 12-13 |
| **Arc welding (250-550A)** | 14 |

## Hearing Protection

### When Required

| Noise Level | Requirement |
|-------------|-------------|
| **85 dBA TWA** | Hearing conservation program |
| **90 dBA TWA** | Hearing protection required |
| **Rig floor** | Always recommended |
| **Near running engines** | Required |

### Types

| Type | NRR | Best For |
|------|-----|----------|
| **Foam plugs** | 25-33 | General use, disposable |
| **Reusable plugs** | 20-27 | Repeated use |
| **Banded plugs** | 20-25 | Intermittent exposure |
| **Earmuffs** | 20-30 | Easy on/off, cold weather |
| **Double protection** | Combined | Very high noise (>100 dBA) |

---

## Hand Protection

### Glove Selection Guide

| Hazard | Glove Type |
|--------|------------|
| **Cut hazards** | Cut-resistant (ANSI A3+) |
| **Impact** | Impact-resistant, back-of-hand |
| **Chemical** | Nitrile, neoprene, or specific |
| **Heat** | Leather or thermal |
| **Cold** | Insulated |
| **General** | Leather work gloves |

### Cut Resistance Levels (ANSI/ISEA 105)

| Level | Grams to Cut | Application |
|-------|--------------|-------------|
| **A1** | 200-499 | Light handling |
| **A2** | 500-999 | General work |
| **A3** | 1000-1499 | Pipe handling |
| **A4** | 1500-2199 | Sharp edges |
| **A5** | 2200-2999 | High cut risk |
| **A6+** | 3000+ | Extreme cut risk |

## Foot Protection

### Requirements

| Feature | Standard |
|---------|----------|
| **Safety toe** | ASTM F2413 |
| **Height** | 6" minimum, 8" preferred |
| **Sole** | Slip-resistant, oil-resistant |
| **Metatarsal** | Required for crushing hazards |
| **Electrical** | EH rated for electrical work |

### Boot Care

| Action | Frequency |
|--------|-----------|
| **Inspect** | Daily |
| **Clean** | As needed |
| **Condition leather** | Monthly |
| **Replace** | When worn or damaged |

## Fire Resistant Clothing (FRC)

### Requirements

| Standard | NFPA 2112 |
|----------|-----------|
| **Arc rating** | 8 cal/cmÂ² minimum |
| **Fit** | No skin exposed during movement |
| **Condition** | No holes, tears, or contamination |
| **Layering** | All underlayers must be FR or natural fiber |

### FRC Rules

| Do | Don't |
|-----|-------|
| Keep clean | Wear contaminated FRC |
| Repair minor damage | Wear synthetic underwear |
| Button/zip fully | Roll up sleeves on rig floor |
| Wash regularly | Bleach or fabric softener |
| Replace if damaged | Ignore damaged garments |

### Arc Flash Categories

| Category | Cal/cmÂ² | Typical Requirement |
|----------|---------|---------------------|
| **1** | 4 | Basic arc-rated shirt/pants |
| **2** | 8 | Standard FRC (well site minimum) |
| **3** | 25 | Flash suit coat/pants |
| **4** | 40 | Full flash suit with hood |

## Respiratory Protection

### When Required

| Condition | Respiratory Protection |
|-----------|----------------------|
| **H2S potential** | Escape packs accessible |
| **H2S > 10 ppm** | SCBA for work |
| **Chemical exposure** | Per SDS and monitoring |
| **Dust/particulates** | N95 or P100 |
| **IDLH atmosphere** | SCBA only |

### Respirator Types

| Type | Protection |
|------|------------|
| **N95** | Particulates (non-oil) |
| **P100** | Particulates (oil resistant) |
| **Half-face APR** | Cartridge-specific chemicals |
| **Full-face APR** | Higher protection, eye coverage |
| **PAPR** | Powered, extended use |
| **SCBA** | IDLH atmospheres, rescue |
| **SAR** | Supplied air, extended duration |

### Fit Testing

| Requirement | Frequency |
|-------------|-----------|
| **Initial fit test** | Before first use |
| **Annual fit test** | Every 12 months |
| **Change in respirator** | New fit test |
| **Weight change** | May require new test |
| **Medical clearance** | Before fit testing |

## Specialized PPE

### Fall Protection

| Equipment | Requirement |
|-----------|-------------|
| **Full body harness** | ANSI Z359.11 |
| **Lanyard** | 6 ft max, shock absorbing |
| **Anchor point** | 5,000 lb capacity |
| **SRL** | ANSI Z359.14 |

### Chemical PPE

| Hazard Level | Protection |
|--------------|------------|
| **Splash hazard** | Goggles, face shield, gloves, apron |
| **Immersion hazard** | Chemical suit |
| **Vapor hazard** | Respirator + suit if needed |

## PPE Inspection

### Daily/Before Use

- [ ] No visible damage
- [ ] Clean and functional
- [ ] Proper fit
- [ ] All components present
- [ ] Within service life

### Documented Inspections

| PPE | Inspection Frequency |
|-----|---------------------|
| **Fall protection** | Before each use + annual formal |
| **SCBA** | Before each use + monthly formal |
| **Respirators** | Before each use |
| **FRC** | Before each use |

---

## PPE Training

### Required Training

| Topic | Content |
|-------|---------|
| **When required** | Hazard assessment |
| **How to use** | Donning, adjustment |
| **Limitations** | What it does and doesn't protect |
| **Care** | Cleaning, storage |
| **Replacement** | When and how |

---

## PPE Checklist

### Before Shift
- [ ] Hard hat inspected
- [ ] Safety glasses clean, undamaged
- [ ] FRC in good condition
- [ ] Proper gloves available
- [ ] Boots in good condition
- [ ] Hearing protection available
- [ ] Specialized PPE ready if needed

### During Work
- [ ] PPE worn correctly
- [ ] PPE appropriate for task
- [ ] PPE adjusted for fit
- [ ] Damaged PPE replaced
