# Hazard Management

## Overview

Hazard management involves systematic identification, assessment, and control of workplace hazards. This includes permit-to-work systems, hot work controls, confined space entry, and lockout/tagout procedures.

---

## Permit to Work System

### When Required

| Activity | Permit Required |
|----------|-----------------|
| **Hot work** | Always |
| **Confined space entry** | Always |
| **Electrical work** | Live or isolation |
| **Excavation** | Breaking ground |
| **Working at heights** | Above 6 ft |
| **Lifting operations** | Critical lifts |
| **Energy isolation** | LOTO situations |

### Permit Elements

| Element | Purpose |
|---------|---------|
| **Description of work** | What is being done |
| **Location** | Specific area |
| **Duration** | Valid time period |
| **Hazards** | Identified risks |
| **Controls** | Required safeguards |
| **Isolation requirements** | Energy/process isolation |
| **PPE** | Required protection |
| **Authorization** | Competent person signature |
| **Acceptance** | Performer acknowledgment |
| **Completion** | Signed off when done |

## Hot Work

### Definition

Any work producing spark, flame, or heat capable of igniting flammable materials

| Hot Work Examples |
|-------------------|
| Welding |
| Cutting (torch, grinder) |
| Brazing |
| Grinding/abrasive cutting |
| Heat treating |
| Thawing |

### Hot Work Requirements

| Requirement | Details |
|-------------|---------|
| **Gas testing** | Test for flammables before and during |
| **Fire watch** | During and 30 min after |
| **Fire extinguisher** | Within 50 ft |
| **Combustible removal** | 35 ft radius minimum |
| **Barriers/blankets** | Protect what can't be moved |
| **Ventilation** | Adequate air flow |
| **Permit** | Signed and posted |

### Hot Work Checklist

**Before Work**:
- [ ] Hot work permit issued
- [ ] Area inspected (35 ft radius)
- [ ] Combustibles removed or protected
- [ ] Gas test performed (<10% LEL)
- [ ] Fire extinguisher present
- [ ] Fire watch designated
- [ ] Ventilation adequate
- [ ] Communication established

**During Work**:
- [ ] Continuous gas monitoring
- [ ] Fire watch observing
- [ ] Welding screens in place
- [ ] No change in conditions

**After Work**:
- [ ] Fire watch continues 30 minutes
- [ ] Area inspected for hot spots
- [ ] Permit closed

## Confined Space Entry

### Confined Space Definition

A space that:
1. Is large enough for a worker to enter
2. Has limited entry/exit
3. Is not designed for continuous occupancy

### Permit-Required Confined Space

Has one or more of:
- Hazardous atmosphere (potential or actual)
- Material that could engulf entrant
- Converging walls or sloping floor
- Other recognized serious hazard

### Confined Space Examples (Oilfield)

| Location | Hazards |
|----------|---------|
| **Tanks** | Vapors, oxygen deficiency, engulfment |
| **Vessels** | Residual product, inert atmosphere |
| **Pits/cellars** | H2S accumulation, oxygen deficiency |
| **Pipeline** | Residual vapors, restricted movement |
| **Bell nipple** | Gas accumulation |

### Entry Requirements

| Requirement | Purpose |
|-------------|---------|
| **Atmospheric testing** | O2, LEL, toxics before and during |
| **Ventilation** | Maintain safe atmosphere |
| **Entry permit** | Document controls |
| **Attendant** | Continuous monitoring from outside |
| **Rescue plan** | Trained rescue team available |
| **Communication** | Constant contact with attendant |
| **Entry log** | Track who's inside |

### Atmospheric Requirements

| Parameter | Safe Range |
|-----------|------------|
| **Oxygen** | 19.5% - 23.5% |
| **LEL** | < 10% |
| **H2S** | < 10 ppm |
| **CO** | < 35 ppm |

### Confined Space Entry Procedure

| Step | Action |
|------|--------|
| 1 | Obtain permit |
| 2 | Isolate all energy sources |
| 3 | Clean/purge space |
| 4 | Test atmosphere |
| 5 | Establish ventilation |
| 6 | Position attendant |
| 7 | Verify rescue capability |
| 8 | Enter with continuous monitoring |
| 9 | Maintain communication |
| 10 | Exit and close permit |

## Lockout/Tagout (LOTO)

### Purpose

Prevent unexpected energization or release of stored energy during maintenance

### Energy Sources

| Energy Type | Examples |
|-------------|----------|
| **Electrical** | Motors, switchgear, batteries |
| **Hydraulic** | Cylinders, accumulators |
| **Pneumatic** | Air systems, bladder tanks |
| **Mechanical** | Springs, raised weights, rotating |
| **Thermal** | Hot surfaces, steam |
| **Chemical** | Pressurized vessels, reactive |
| **Gravity** | Elevated equipment, suspended loads |

### LOTO Sequence

```
┌─────────────────────────────────────────────────────────────────┐
│                       LOTO SEQUENCE                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1. NOTIFY affected personnel                                  │
│                │                                                │
│                ▼                                                │
│   2. IDENTIFY all energy sources                               │
│                │                                                │
│                ▼                                                │
│   3. ISOLATE equipment from energy                             │
│                │                                                │
│                ▼                                                │
│   4. LOCK and TAG isolation points                             │
│                │                                                │
│                ▼                                                │
│   5. RELEASE stored/residual energy                            │
│                │                                                │
│                ▼                                                │
│   6. VERIFY zero energy (try start)                            │
│                │                                                │
│                ▼                                                │
│   7. PERFORM the work                                          │
│                │                                                │
│                ▼                                                │
│   8. RESTORE (reverse sequence with authorization)             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### LOTO Requirements

| Requirement | Details |
|-------------|---------|
| **Personal lock** | Each worker applies own lock |
| **Tags** | Identify worker, date, purpose |
| **One key only** | No master keys |
| **Verification** | Try to start after lockout |
| **Authorization** | Supervisor approves removal |

## Working at Heights

### When Fall Protection Required

| Height | Requirement |
|--------|-------------|
| **6 ft and above** | Fall protection required |
| **On leading edge** | Fall protection required |
| **Near holes/openings** | Fall protection or covers |
| **On scaffolding** | Per scaffold requirements |

### Fall Protection Methods

| Method | Application |
|--------|-------------|
| **Guardrails** | Permanent protection |
| **Safety nets** | Below work area |
| **Personal fall arrest** | Harness and lanyard |
| **Positioning devices** | Work positioning |
| **Warning lines** | Roofing work |

### Personal Fall Arrest System

| Component | Requirement |
|-----------|-------------|
| **Full body harness** | Inspected before each use |
| **Lanyard** | Shock absorbing, proper length |
| **Anchor point** | 5,000 lb capacity per person |
| **Self-retracting lifeline** | Alternative to lanyard |

### Fall Protection Inspection

| Check | Frequency |
|-------|-----------|
| **Visual inspection** | Before each use |
| **Documented inspection** | Annually |
| **After fall event** | Remove from service |

## Lifting Operations

### Critical Lift Definition

A lift that:
- Exceeds 75% of crane capacity
- Requires more than one crane
- Is over personnel or live equipment
- Involves hazardous materials
- Has unusual rigging

### Lift Planning

| Element | Details |
|---------|---------|
| **Load weight** | Known, verified |
| **Rigging** | Rated capacity, inspected |
| **Crane capacity** | Adequate at radius |
| **Ground conditions** | Stable, level |
| **Overhead hazards** | Power lines, structures |
| **Wind** | Within limits |
| **Communication** | Signals, radio |

### Pre-Lift Checklist

- [ ] Load weight confirmed
- [ ] Rigging inspected and rated
- [ ] Crane capacity verified at radius
- [ ] Tag lines attached
- [ ] Exclusion zone established
- [ ] Signals agreed
- [ ] Spotter positioned
- [ ] Conditions acceptable (wind, weather)

## Hazard Identification

### HAZID/HAZOP

| Method | Application |
|--------|-------------|
| **HAZID** | Hazard identification for new projects |
| **HAZOP** | Hazard and operability study for processes |

### Risk Assessment Matrix

| Probability / Severity | Minor | Moderate | Major | Catastrophic |
|----------------------|-------|----------|-------|--------------|
| **Frequent** | Medium | High | Extreme | Extreme |
| **Probable** | Low | Medium | High | Extreme |
| **Occasional** | Low | Medium | High | High |
| **Remote** | Low | Low | Medium | High |
| **Improbable** | Low | Low | Low | Medium |

### Hierarchy of Controls

```
┌─────────────────────────────────────────┐
│            ELIMINATION                  │ Most Effective
│       (Remove the hazard)               │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐

└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐

                  │

                  │
```
