## Key Concepts

### Safety Hierarchy

```
        ┌─────────────────┐
        │   ELIMINATION   │  ← Most Effective
        │  Remove hazard  │
        └────────┬────────┘
                 │
        ┌────────▼────────┐
        │  SUBSTITUTION   │
        │ Replace hazard  │
        └────────┬────────┘
                 │
        ┌────────▼────────┐
        │   ENGINEERING   │
        │ Isolate people  │
        └────────┬────────┘
                 │
        ┌────────▼────────┐
        │ ADMINISTRATIVE  │
        │ Change behavior │
        └────────┬────────┘
                 │
        ┌────────▼────────┐
        │       PPE       │  ← Least Effective
        │ Protect worker  │
        └─────────────────┘
```

### HSE Priorities

| Priority | Focus | Examples |
|----------|-------|----------|
| **1. Life Safety** | Protect human life | H2S, well control, fall protection |
| **2. Environment** | Prevent pollution | Spill prevention, emissions, waste |
| **3. Assets** | Protect equipment | Fire prevention, maintenance |
| **4. Production** | Maintain operations | Only after 1-3 are secured |

### Key HSE Metrics

| Metric | Definition | Target |
|--------|------------|--------|
| **TRIR** | Total Recordable Incident Rate | < 1.0 |
| **DART** | Days Away/Restricted/Transfer | < 0.5 |
| **Severity Rate** | Lost days per 200,000 hours | < 5.0 |
| **Near Miss Ratio** | Near misses reported per incident | > 10:1 |

### Critical Success Factors

1. **Leadership commitment** - Safety starts at the top
2. **Employee engagement** - Everyone participates
3. **Hazard identification** - Find them before they find you
4. **Training** - Competent, qualified workers
5. **Continuous improvement** - Learn from every incident
