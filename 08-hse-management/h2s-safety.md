# H2S Safety

## Overview

Hydrogen Sulfide (H2S) is a colorless, highly toxic gas found in many oil and gas operations. H2S exposure can be immediately fatal. Every person on location must understand H2S hazards and emergency procedures.

---

## H2S Properties

### Physical Characteristics

| Property | Value |
|----------|-------|
| **Chemical formula** | H₂S |
| **Molecular weight** | 34.08 |
| **Color** | Colorless |
| **Odor** | Rotten egg (at low concentrations) |
| **Specific gravity** | 1.189 (heavier than air) |
| **Flammable range** | 4.3% - 46% |
| **Solubility** | Soluble in water |

## Exposure Limits

### Regulatory Limits

| Standard | Concentration | Exposure |
|----------|---------------|----------|
| **OSHA PEL** | 20 ppm | Ceiling (never exceed) |
| **OSHA STEL** | 50 ppm | 10 minutes max |
| **NIOSH REL** | 10 ppm | 10 minutes max |
| **IDLH** | 100 ppm | Immediately Dangerous to Life/Health |

### Health Effects by Concentration

| Concentration (ppm) | Effects |
|--------------------|---------|
| **0.01 - 0.3** | Odor threshold |
| **2 - 5** | Moderate odor, nausea possible |
| **10 - 20** | Eye and respiratory irritation |
| **50 - 100** | Severe eye/respiratory damage |
| **100 - 200** | Olfactory fatigue (can't smell), serious injury |
| **200 - 300** | Marked respiratory effects |
| **500 - 700** | Collapse, unconsciousness |
| **700 - 1000** | Immediate death possible |
| **1000+** | Instant death |

## H2S Detection

### Fixed Monitors

| Location | Purpose |
|----------|---------|
| **Shale shaker** | Detect gas in returns |
| **Mud tanks** | Monitor mud system |
| **BOP/wellhead** | High-risk area |
| **Living quarters** | Personnel protection |
| **Cellar** | Low-lying accumulation |

### Personal Monitors

| Type | Features |
|------|----------|
| **Single gas** | H2S only |
| **Multi-gas** | H2S, LEL, O2, CO |
| **Bump tested** | Before each use |
| **Calibrated** | Per manufacturer schedule |

### Alarm Levels

| Level | Concentration | Action |
|-------|---------------|--------|
| **Low (Alert)** | 10 ppm | Investigate, wear monitor |
| **High (Danger)** | 20 ppm | Don SCBA, evacuate non-essential |
| **IDLH** | 100 ppm | Evacuate all, emergency response |

|-----------|----------|----------|

| **Fixed monitors** | Per plan | Critical areas |
| **Personal monitors** | 1 per person | Worn by all |
| **Rescue equipment** | As required | Designated area |

### SCBA (Self-Contained Breathing Apparatus)

**Duration**: Typically 30-60 minutes

| Inspection | Frequency |
|------------|-----------|
| **Visual** | Before each use |
| **Function test** | Before each use |
| **Flow test** | Monthly |
| **Hydrostatic test** | Per manufacturer |

### Escape-Only Respirator (Escape Pack)

**Duration**: 5-15 minutes (escape only)

| Feature | Requirement |
|---------|-------------|
| **Location** | Worn or within arm's reach |
| **Inspection** | Weekly |
| **Training** | All personnel |
| **Use** | Escape only, NOT for entry |

## H2S Emergency Response

### Detection Response

| Alarm Level | Immediate Action |
|-------------|------------------|
| **10 ppm** | Alert, investigate, monitor |
| **20 ppm** | Don breathing apparatus, limit personnel |
| **100 ppm** | Evacuate ALL personnel immediately |

### Rescue Procedures

**CRITICAL**: Only trained rescuers with SCBA may attempt rescue

| Step | Action |
|------|--------|
| 1 | Don SCBA (buddy system) |
| 2 | Communicate with supervisor |
| 3 | Approach upwind |
| 4 | Remove victim to fresh air |
| 5 | Begin first aid/CPR if needed |
| 6 | Seek medical attention |

### First Aid for H2S Exposure

| Condition | Action |
|-----------|--------|
| **Conscious, breathing** | Move to fresh air, monitor |
| **Unconscious, breathing** | Move to fresh air, recovery position, monitor |
| **Not breathing** | CPR immediately, continue until EMS |
| **Convulsions** | Protect from injury, do not restrain |
