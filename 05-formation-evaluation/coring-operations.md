# Coring Operations

## Overview

Coring provides physical rock samples from the formation for laboratory analysis. Core data provides ground truth for log interpretation and detailed reservoir characterization.

---

## Core Types

### Conventional Coring

**Description**: Full diameter cores cut while drilling with special core barrel

| Specification | Common Sizes |
|---------------|--------------|
| **Core diameter** | 2.5", 4", 5.25" |
| **Core length** | 30 ft, 60 ft, 90 ft per barrel |
| **Hole size** | 6.25" - 12.25" |

**Advantages**:
- Large sample volume
- Preserved structure
- Complete sequence
- Multiple analysis types

**Disadvantages**:
- Expensive (trip pipe)
- Time consuming
- Risk of leaving core in hole
- Formation damage possible

### Sidewall Coring

**Percussion (Bullet)**:
- Small samples (1" × 1.5")
- 20-60 samples per run
- Quick, wireline deployed
- Can damage soft formations

**Rotary Sidewall**:
- Larger samples (1" × 2")
- Better sample quality
- Less formation damage
- More expensive

| Feature | Percussion | Rotary |
|---------|------------|--------|
| Sample size | Smaller | Larger |
| Sample quality | May be damaged | Better preserved |
| Depth limit | 20,000+ ft | 20,000+ ft |
| Hard formations | Poor recovery | Good recovery |
| Unconsolidated | Good | Fair |
| Time | Faster | Slower |

## Coring Parameters

### Drilling Parameters for Coring

| Parameter | Range | Notes |
|-----------|-------|-------|
| **WOB** | 2,000-8,000 lb | Lower than normal drilling |
| **RPM** | 40-100 | Controlled rotation |
| **Flow rate** | 50-75% of normal | Prevent washing core |
| **ROP** | Control to match formation | Typically 15-60 ft/hr |

### Coring Best Practices

| Practice | Purpose |
|----------|---------|
| **Slow, steady WOB** | Prevent jamming, grinding |
| **Constant RPM** | Even cutting |
| **Reduced flow** | Prevent core washing |
| **Monitor torque** | Detect jamming |
| **Limit inner barrel speed** | Reduce grinding |
| **Core carefully in unconsolidated** | Prevent core loss |

## Core Recovery and Handling

### At Surface Procedures

1. **Lay down carefully** - Avoid shock/vibration
2. **Remove from barrel systematically** - Maintain orientation
3. **Mark immediately** - Top, bottom, depth, orientation
4. **Photograph** - Fresh surfaces, UV fluorescence
5. **Describe** - Lithology, shows, features
6. **Preserve** - Proper containers, wax if needed
7. **Ship promptly** - Minimize exposure

### Core Marking Convention

```
TOP OF CORE (Shallower)
          │
          ▼
    ┌─────────────────┐
    │  DEPTH: 8,500'  │ ← Mark depth at each break
    │  ▼ ▼ ▼ ▼ ▼ ▼ ▼  │ ← Arrows point DOWN
    │                 │
    │  RED LINE ──────┼─→ Orientation line (if oriented)
    │                 │
    │                 │
    │  ▼ ▼ ▼ ▼ ▼ ▼ ▼  │
    │  DEPTH: 8,530'  │
    └─────────────────┘
          │
          ▼
BOTTOM OF CORE (Deeper)
```

### Core Description Format

| Element | Description |
|---------|-------------|
| **Depth interval** | Top to bottom of section |
| **Lithology** | Rock type, color |
| **Sedimentary features** | Bedding, laminations |
| **Porosity type** | Visible porosity |
| **Fractures** | Natural, induced |
| **Shows** | Oil staining, fluorescence, odor |
| **Fossils** | If present |
| **Recovery** | Percentage recovered |

## Core Analysis Program

### Routine Core Analysis (RCAL)

| Measurement | Purpose |
|-------------|---------|
| **Porosity** | Storage capacity |
| **Permeability** | Flow capacity |
| **Grain density** | Lithology, log calibration |
| **Fluid saturations** | Oil, water, gas content |

### Special Core Analysis (SCAL)

| Measurement | Purpose |
|-------------|---------|
| **Relative permeability** | Multi-phase flow |
| **Capillary pressure** | Fluid distribution |
| **Wettability** | Fluid preference |
| **Electrical properties** | Log calibration (m, n) |
| **Compressibility** | Reservoir compaction |
| **Acoustic properties** | Sonic log calibration |

### Core Preservation

| Preservation Method | Application |
|---------------------|-------------|
| **Wax coating** | Prevent fluid loss, short-term |
| **Plastic wrap** | Maintain moisture |
| **Freezing** | Unconsolidated cores |
| **Aluminum containers** | Long-term storage |
| **Nitrogen purge** | Prevent oxidation |

## Coring Problems

### Common Problems and Solutions

| Problem | Cause | Solution |
|---------|-------|----------|
| **Jamming** | Hard streak, rubble | Reduce WOB, increase RPM |
| **Low recovery** | Unconsolidated, fractured | Better core catcher, adjust parameters |
| **Core washing** | Excessive flow | Reduce pump rate |
| **Grinding** | Too much WOB, low RPM | Adjust parameters |
| **Stuck core barrel** | Differential sticking, jammed | Work pipe, jar if available |
| **Core left in hole** | Core catcher failure | Fish for core, redrill |

### Recovery Calculation

```
Core Recovery (%) = (Core Recovered Length / Cored Interval Length) × 100

Example:
Cored from 8,500 to 8,560 ft (60 ft interval)
Core recovered = 52 ft
Recovery = 52/60 × 100 = 86.7%
```

- Fracture orientation studies
- Stress analysis
- Paleocurrent analysis

|--------|----------|-------------|

| **Electronic multishot** | ±3° | High accuracy |
| **Gyro orientation** | ±1° | Highest accuracy |

## Sponge Coring

### Purpose
- Capture and preserve reservoir fluids
- Measure residual oil saturation
- Used in EOR projects

### Mechanism
- Inner barrel lined with oil-absorbing sponge
- Fluids expelled from core absorbed by sponge
- Sponge analyzed along with core

## Coring Decision Factors

### When to Core

| Factor | Consider Coring If |
|--------|-------------------|
| **Exploration well** | Always in pay zone |
| **Reservoir uncertainty** | Need ground truth |
| **Log ambiguity** | Unclear lithology or fluid |
| **Special studies** | Fractures, geomechanics |
| **Enhanced recovery** | SCAL required |
| **Regulatory** | Some permits require |

### Core vs. No Core Decision

| Core | Don't Core |
|------|------------|
| First well in new field | Development well with good offset data |
| Unknown reservoir | Well-characterized reservoir |
| Need petrophysical calibration | Good log-core relationships exist |
| Complex lithology | Simple, uniform formation |
| Fracture characterization needed | Matrix-dominated reservoir |

## Coring Checklist

### Pre-Coring
- [ ] Core barrel inspected and tested
- [ ] Core catchers appropriate for formation
- [ ] Core handling equipment ready
- [ ] Core description forms prepared
- [ ] Sample containers and preservatives ready
- [ ] Lab notified of incoming cores
- [ ] Core point and interval confirmed

### During Coring
- [ ] Monitor WOB, RPM, torque, ROP
- [ ] Note any indications of jamming
- [ ] Record coring parameters
- [ ] Track depth carefully
- [ ] Note shows from circulated cuttings

### After Coring
- [ ] Handle core carefully at surface
- [ ] Mark core immediately
- [ ] Photograph and describe
- [ ] Preserve appropriately
- [ ] Complete core report
- [ ] Ship to lab promptly
