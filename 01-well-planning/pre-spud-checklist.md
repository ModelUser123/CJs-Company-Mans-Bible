# Pre-Spud Checklist

## Overview

The pre-spud checklist ensures all preparations are complete before drilling operations commence. This is the final verification that the well is ready to spud.

---

## Administrative Preparations

### Documentation Complete

```
□ AFE approved and distributed
□ Drilling permit received
□ Environmental permits in place
□ Surface use agreement signed
□ Road use permits obtained
□ Water disposal permits
□ Air permits (if applicable)
□ Notification to regulatory agency
```

### Contracts and Insurance

```
□ Rig contract executed
□ All service contracts in place
□ Insurance certificates collected
□ Indemnity agreements signed
□ Well control insurance confirmed
□ Pollution liability coverage verified
```

### Well Program Distribution

```
□ Drilling program finalized
□ Mud program approved
□ Casing program approved
□ Cementing program approved
□ Directional program approved
□ Logging program approved
□ All programs distributed to contractors
```

## Pre-Spud Meeting

### Agenda

1. **Well Objectives**
   - Target zones
   - Depth objectives
   - Expected challenges

2. **Drilling Program Overview**
   - Hole sizes and casing points
   - Mud weights and types
   - Estimated times

3. **Safety Focus**
   - Specific hazards
   - Mitigation measures
   - STOP work authority

4. **Well Control**
   - BOP equipment
   - Kill procedure review
   - Kick detection

5. **Communication**
   - Reporting requirements
   - Chain of command
   - Emergency contacts

6. **Contractor Coordination**
   - Roles and responsibilities
   - Interface points
   - Resource sharing

7. **Environmental**
   - Spill prevention
   - Waste management
   - Wildlife concerns

### Required Attendees

- Company Man
- Drilling Superintendent (or representative)
- Toolpusher
- Drillers (both tours)
- Directional Driller
- Mud Engineer
- Mud Logger
- Safety Representative
- Environmental Representative
- Key service company supervisors

### Documentation

```
□ Sign-in sheet completed
□ Meeting minutes recorded
□ Action items assigned
□ Program distribution confirmed
□ Questions/concerns documented
```

---

## Final Verification

### 24 Hours Before Spud

```
□ Weather forecast acceptable
□ All permits posted
□ All personnel on location
□ All equipment functional
□ All materials on location
□ Communications working
□ Emergency response ready
□ Pre-spud meeting complete
```

### Day of Spud

```
□ Morning safety meeting held
□ JSA completed for surface hole
□ Equipment final check
□ Mud system charged
□ BOP tested (pressure)
□ All hands briefed
□ Spud time recorded
```

## Spud Report Requirements

### Notification List

- Regulatory agency (as required)
- Company operations center
- Partner representatives
- Insurance underwriter
- Service companies

### Initial Report Content

- Well name and number
- Location coordinates
- Spud date and time
- Rig contractor and rig name
- Company Man name and contact
- Initial operations summary
