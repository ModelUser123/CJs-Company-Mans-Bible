# Frac Diagnostics

## Overview

Frac diagnostics help evaluate treatment effectiveness, optimize future designs, and understand fracture behavior. Multiple diagnostic methods provide different information about the hydraulic fracturing process.

---

## Pressure-Based Diagnostics

### DFIT (Diagnostic Fracture Injection Test)

**Purpose**: Pre-frac determination of stress, pressure, permeability

**Procedure**:
1. Inject small volume at low rate
2. Shut in and monitor decline
3. Analyze pressure-time data

**Key Parameters**:

| Parameter | Source |
|-----------|--------|
| **Closure pressure (Pc)** | G-function derivative |
| **Initial reservoir pressure** | Extrapolation |
| **Permeability** | After-closure analysis |
| **Leakoff coefficient** | Before-closure analysis |

### G-Function Analysis

```
G(ΔtD) = (4/π) × [g(ΔtD) - g(0)]

Where:
ΔtD = Dimensionless time = Δt/tp
tp = Pump time
Δt = Shut-in time
```

**Identification of Closure**:
- Plot pressure vs. G-time
- Closure at departure from linearity
- GdP/dG identifies mechanism

### Pressure Decline Analysis

| Observation | Interpretation |
|-------------|----------------|
| **Linear on sqrt(t)** | Normal leakoff |
| **Concave up** | Pressure-dependent leakoff |
| **Concave down** | Height recession |
| **Multiple slopes** | Changing mechanisms |

## Net Pressure Analysis

### Net Pressure Definition

```
Net Pressure = ISIP - Closure Pressure

Pnet = PISIP - Pc
```

### Nolte-Smith Plot

**Log(Pnet) vs. Log(time)**

| Slope | Interpretation |
|-------|----------------|
| **1/4 to 1/8** | Normal PKN extension |
| **Flat (0)** | Height growth |
| **Negative** | Uncontrolled height, tip screenout |
| **Steep (> 1/4)** | Near-wellbore problems |
| **Rising then flat** | Tip screenout (intended) |

### Treatment Pressure Interpretation

| Pattern | Likely Cause |
|---------|--------------|
| **Steady treatment pressure** | Good propagation |
| **Rising pressure** | Approaching screenout |
| **Declining pressure** | Height growth or out of zone |
| **Erratic pressure** | Equipment issues or bridging |

## Microseismic Monitoring

### Concept

**Microseismic events**: Small earthquakes generated by rock fracturing

```
┌─────────────────────────────────────────────────────────────────┐
│              MICROSEISMIC MONITORING SETUP                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Monitor Well        Treatment Well                            │
│       │                    │                                    │
│       │                    │                                    │
│       │   ┌────────────────┼──────────────────┐                │
│       │   │                │                  │                │
│       ▼   │                ▼                  │                │
│   ┌───────┴───┐    ════════════════════      │                │
│   │  Geophone │    ●  ●   ●  ●  ●   ●       │                │
│   │  Array    │      ●  ●   ●  ●            │                │
│   │           │    ●  ●   Microseismic      │                │
│   │           │      ●      Events          │                │
│   └───────────┘                              │                │
│                                              │                │
└─────────────────────────────────────────────────────────────────┘
```

### Microseismic Outputs

| Output | Information |
|--------|-------------|
| **Event locations** | Fracture geometry |
| **Event timing** | Fracture growth rate |
| **Event magnitude** | Energy release |
| **Stress regime** | Fracture orientation |
| **SRV (Stimulated Rock Volume)** | Treatment extent |

### Interpretation Guidelines

| Observation | Interpretation |
|-------------|----------------|
| **Linear event cloud** | Planar fracture |
| **Wide event cloud** | Complex fractures |
| **Height growth** | Events above/below target |
| **Cluster activation** | Near-perf complexity |
| **Time progression** | Fracture velocity |

### Limitations

- Location uncertainty (10-50 ft typical)
- Not all fractures generate events
- Requires nearby monitor well or surface array
- Cost significant

## Fiber Optic Diagnostics

### Distributed Acoustic Sensing (DAS)

**Uses fiber optic cable to measure acoustic signals along entire wellbore**

| Application | Information |
|-------------|-------------|
| **During frac** | Fluid distribution, cluster efficiency |
| **Flowback** | Production profile |
| **Long-term** | Changes over time |

### Distributed Temperature Sensing (DTS)

**Measures temperature profile along wellbore**

| Application | Information |
|-------------|-------------|
| **Warmback analysis** | Which clusters took fluid |
| **Production** | Flow profile |
| **Cross-flow** | Unwanted communication |

### Combined DAS/DTS Benefits

| Benefit | Description |
|---------|-------------|
| **Cluster efficiency** | Which clusters contribute |
| **Stage coverage** | Fluid placement verification |
| **Production allocation** | Stage-by-stage contribution |
| **Optimization** | Design future wells |

## Production-Based Diagnostics

### Rate Transient Analysis (RTA)

**Analyze production data to estimate fracture properties**

| Method | Output |
|--------|--------|
| **Flowing material balance** | Drainage volume, k×h |
| **Type curve matching** | Fracture half-length, conductivity |
| **Decline curve analysis** | EUR, b-factor |

### Production Logging

| Log | Information |
|-----|-------------|
| **Spinner** | Flow velocity |
| **Temperature** | Fluid entry |
| **Noise** | Active perforations |
| **Water holdup** | Phase distribution |

## Tracer Surveys

### Chemical Tracers

| Tracer Type | Application |
|-------------|-------------|
| **Water-soluble** | Track frac fluid |
| **Oil-soluble** | Track hydrocarbon production |
| **Proppant-borne** | Track proppant placement |

### Tracer Analysis

| Measurement | Information |
|-------------|-------------|
| **Which wells return tracer** | Inter-well communication |
| **Which stages return tracer** | Stage contribution |
| **Recovery time** | Fracture connectivity |
| **Concentration profile** | Fracture complexity |

### Radioactive Tracers

- Tagged proppant or fluid
- Gamma log after treatment
- Shows tracer location in wellbore
- Indicates placement

## Frac Hit Monitoring

### What is a Frac Hit?

**Communication between treatment well and offset (parent) well**

### Monitoring Parent Wells

| Monitor | Watch For |
|---------|-----------|
| **Pressure** | Increase during child frac |
| **Rate changes** | Decrease or increase |
| **Water cut** | Increase |
| **GOR** | Changes |
| **Microseismic** | Events near parent |

### Frac Hit Mitigation

| Strategy | Method |
|----------|--------|
| **Pre-loading** | Pump into parent before frac |
| **Pressure management** | Maintain parent pressure |
| **Spacing optimization** | Increase well spacing |
| **Sequencing** | Zipper, timing of child fracs |

## Completion Efficiency Metrics

### Cluster Efficiency

```
Cluster Efficiency = Active Clusters / Total Clusters × 100%

Goal: > 80-90% efficiency
```

### Stage Contribution

**Production allocation by stage**:
- Use tracers, PLT, or DAS
- Identify underperforming stages
- Guide refrac decisions

### Treatment Effectiveness

| Metric | Calculation |
|--------|-------------|
| **EUR/ft** | EUR / Lateral length |
| **EUR/stage** | EUR / Number of stages |
| **EUR/proppant** | EUR / Proppant mass |
| **Production/$ invested** | Production / Treatment cost |

## Post-Treatment Analysis

### Data Collection

| Data | Source |
|------|--------|
| **Treating data** | Frac van records |
| **Production data** | Daily production |
| **Pressure data** | Wellhead and BHP |
| **Diagnostic results** | Microseismic, DAS, etc. |
| **Offset data** | Comparison wells |

### Analysis Workflow

1. **Collect all data**
2. **Pressure analysis** (closure, efficiency)
3. **Production analysis** (type curves, RTA)
4. **Compare to design** (actual vs. predicted)
5. **Identify learnings** (what worked, what didn't)
6. **Update future designs** (continuous improvement)

### Key Questions

| Question | Why Important |
|----------|---------------|
| **Did we achieve design fracture length?** | Treatment effectiveness |
| **What was cluster efficiency?** | Completion optimization |
| **How does production compare?** | Economic success |
| **What can we improve?** | Next well optimization |

## Diagnostic Selection Guide

| Objective | Recommended Diagnostics |
|-----------|------------------------|
| **Pre-frac stress** | DFIT |
| **Fracture geometry** | Microseismic, tiltmeter |
| **Cluster efficiency** | DAS/DTS, tracers |
| **Production profile** | PLT, RTA |
| **Frac hits** | Parent well monitoring |
| **Treatment optimization** | All above + history matching |
