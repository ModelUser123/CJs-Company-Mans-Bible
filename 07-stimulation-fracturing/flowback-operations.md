# Flowback Operations

## Overview

Flowback is the process of recovering fracturing fluid after stimulation and transitioning to production. Proper flowback management protects the proppant pack, maximizes fluid recovery, and establishes optimal production.

---

## Flowback Objectives

### Primary Goals

| Objective | Why Important |
|-----------|---------------|
| **Recover frac fluid** | Reduce formation damage |
| **Clean up fractures** | Remove residual gel, debris |
| **Protect proppant pack** | Prevent crushing, flowback |
| **Establish production** | Transition to sales |
| **Collect data** | Evaluate treatment success |

## Flowback Procedures

### Initial Flowback

**Opening Sequence**:
1. Verify all equipment and connections
2. Establish monitoring
3. Slowly open choke (1/64" increments)
4. Monitor pressure response
5. Establish stable flow

### Controlled Flowback

| Parameter | Guideline |
|-----------|-----------|
| **Initial choke** | 4-8/64" typical |
| **Rate increase** | Gradual, step-wise |
| **Max drawdown** | Per design, protect proppant |
| **Monitoring** | Continuous pressure, rate, sand |

### Choke Management Program

**Example Schedule**:

| Time | Choke Size | Target Rate |
|------|------------|-------------|
| Hour 0-6 | 8/64" | Initial flow |
| Hour 6-12 | 12/64" | Increase |
| Hour 12-24 | 16/64" | Build rate |
| Day 2-3 | 20/64" | Approach target |
| Day 3+ | As needed | Optimize |

## Proppant Pack Protection

### Key Concepts

**Closure Stress**: Formation stress trying to crush proppant
**Embedment**: Proppant pressing into formation face
**Flowback Risk**: Proppant produced to surface

### Protection Guidelines

| Guideline | Reason |
|-----------|--------|
| **Slow initial flowback** | Allow closure, proppant setting |
| **Avoid rapid pressure changes** | Prevent pack disturbance |
| **Monitor sand production** | Early warning |
| **Control drawdown** | Limit stress changes |

### Critical Flowback Rate

```
CFR = Rate above which proppant flowback begins

Stay below CFR during early flowback
```

### Sand Production Monitoring

| Level | Action |
|-------|--------|
| **Trace** | Monitor, continue |
| **Light** | Reduce rate |
| **Heavy** | Shut in, evaluate |
| **Continuous** | Problem, intervention needed |

## Fluid Recovery Tracking

### Recovery Calculation

```
Fluid Recovery (%) = (Recovered Volume / Pumped Volume) × 100

Example:
Pumped: 10,000 bbl frac fluid
Recovered: 4,500 bbl
Recovery = 45%
```

### Typical Recovery Rates

| Formation | Expected Recovery |
|-----------|-------------------|
| **Tight sand** | 20-40% |
| **Shale** | 10-30% |
| **Conventional** | 40-70% |

### Tracking Metrics

| Parameter | Track |
|-----------|-------|
| **Cumulative water** | Total recovered |
| **Daily water** | Rate of recovery |
| **Chlorides** | Formation vs. frac water |
| **pH** | Breakdown of frac fluid |
| **Total dissolved solids** | Transition indicator |

## Gas Flaring

### When Required

- Gas production exceeds handling capacity
- No sales infrastructure
- During cleanup phase
- Safety releases

### Flare Operations

| Consideration | Requirement |
|---------------|-------------|
| **Permit** | Usually required |
| **Location** | Safe distance, downwind |
| **Pilot** | Maintain continuous ignition |
| **Rate** | Design for expected rate |
| **Monitoring** | Continuous observation |

### Environmental Compliance

| Requirement | Details |
|-------------|---------|
| **Reporting** | Volume flared, duration |
| **Duration limits** | Per regulation |
| **Green completions** | Capture if possible |
| **Emissions** | Report as required |

## Troubleshooting

### Common Problems

| Problem | Possible Cause | Solution |
|---------|----------------|----------|
| **No flow** | Plugged choke, screenout | Work choke, coil tubing |
| **Slugging** | Liquid loading, gas interference | Adjust choke, plunger |
| **Heavy sand** | Proppant flowback | Reduce rate, shut in |
| **Low rate** | Damage, poor treatment | Restimulate, evaluate |
| **Excessive water** | Water contact, frac height | Monitor, possible recompletion |
| **Emulsion** | Frac fluid/oil interaction | Demulsifier treatment |

### Proppant Flowback Response

| Action | When |
|--------|------|
| **Reduce rate** | First sign of sand |
| **Cycle** | Open/close may settle |
| **Shut in** | Allow pack to stabilize |
| **Restart slowly** | After shut-in period |
| **Intervention** | If continues |

## Safety Considerations

### Hazards

| Hazard | Mitigation |
|--------|------------|
| **High pressure** | Proper ratings, barriers |
| **H2S** | Monitoring, PPE, training |
| **Flammable gas** | Ignition control, flaring |
| **Chemicals** | PPE, containment |
| **Noise** | Hearing protection |

### Emergency Response

| Scenario | Response |
|----------|----------|
| **High pressure event** | Shut in at wellhead |
| **Fire** | Shut in, evacuate |
| **H2S detection** | Evacuate upwind, breathing air |
| **Spill** | Contain, report |

- [ ] Flare system operational
- [ ] Monitoring systems online

- [ ] Fluid samples collected
- [ ] Sand monitored continuously
- [ ] Water recovered tracked
- [ ] Safety monitoring active
- [ ] Gas flaring per permit

- [ ] Cleanup confirmed complete
- [ ] Production equipment ready

- [ ] Documentation complete
