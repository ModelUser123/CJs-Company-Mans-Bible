# Reporting Requirements

## Daily/Weekly Operations Reports

### Daily Drilling Report (DDR)
```
Required Information:
- Date and report number
- Depth (start, end, total drilled)
- Formation being drilled
- Mud properties (weight, vis, pH, etc.)
- Operations summary by hour
- Equipment status
- Safety statistics (LTI, TRI, etc.)
- Cost data (actual vs AFE)
- Contractor performance
- Problems encountered
- Next 24-hour plan

Submitted to:
- Company drilling superintendent
- Operations management
- Partners (if applicable)
```

### IADC Daily Report
```
Standard industry format including:
- Hole section data
- Drilling parameters
- Fluid data
- BHA description
- Survey data
- Bit records
- Coded activity breakdown
```

### Weekly BOP Test Report
```
Contents:
- Date and time of test
- Type of test (pressure/function)
- Preventer type and size
- Test pressure achieved
- Duration held
- Pass/fail status
- Witnesses
- Repairs made (if any)

BSEE Requirement: Form BSEE-0130
State Requirements: Varies by jurisdiction
```

## Completion and Post-Drilling Reports

### Completion Report
```
Due: Typically 30 days after completion

Contents:
- Final well data (TD, formations)
- Casing record (all strings)
- Cement record (all jobs)
- Perforation details
- Stimulation summary
- Test results
- Production equipment installed
- As-built wellbore diagram
```

### Well Log Submission
```
Logs to Submit:
- Gamma ray
- Resistivity
- Porosity logs
- Cement bond log
- Formation evaluation logs
- Directional surveys

Format:
- LAS files (digital)
- PDF (scans acceptable)
- Paper copies (some agencies)

Due: 30-60 days after completion
```

### Cement Record
```
Required for Each Job:
- Date and time
- Casing size and depth
- Hole size
- Cement type and class
- Slurry weight
- Yield
- Volume pumped
- Displacement volume
- Top of cement (calculated)
- Returns observed
- CBL/VDL results (if run)
```

## Production Reports

### Monthly Production Report
```
Due Date by State:
- Texas: 45 days after month end
- Oklahoma: 15th of following month
- New Mexico: 15th of second following month
- Louisiana: 15th of following month
- North Dakota: 20th of following month

Contents:
- Oil production (barrels)
- Gas production (Mcf)
- Water production (barrels)
- Days on production
- Disposal method
- Well status
```

### Annual Reports
```
Common Annual Reports:
- Financial assurance renewal
- Well status report
- Mechanical integrity test results
- Inactive well status
- Waste disposal summary
- Emergency response plan update
```

## Incident Reports

### Immediate Notification Required
```
BSEE (Offshore):
- Fatality
- Major injury (hospitalization)
- Fire/explosion
- Loss of well control
- Pollution over 1 bbl
- Collision damage
- Structural damage

Report: Phone call + Written follow-up
Timeline: Immediately (within 2 hours)
```

### 24-Hour Reports
```
Incidents requiring 24-hour report:
- Lost time injuries
- Minor fires
- Small spills (<1 bbl offshore)
- Equipment failures affecting safety
- Near-misses with potential for serious injury
```

### Spill Reports

```
Federal (EPA):
- Report spills to NRC if:
  - Reaches navigable waters
  - Exceeds RQ (reportable quantity)
  - Threatens public health
  - Violates permit

NRC: 1-800-424-8802

State Reports:
- Most require immediate notification
- Written follow-up within 24-48 hours
- Cleanup documentation
```

### Root Cause Investigation
```
Required Following:
- Fatalities
- Serious injuries
- Major equipment damage
- Significant environmental impact
- Near-misses (by company policy)

Contents:
- Incident description
- Timeline of events
- Root cause analysis
- Contributing factors
- Corrective actions
- Preventive measures
```

## Survey Reports

### Directional Survey Submission
```
Survey Requirements:
- Surface location verification
- At regular intervals during drilling
- At casing points
- At total depth
- At completion depth

Submit:
- Survey data (MD, Inc, Az, TVD, coordinates)
- Survey tool specifications
- Quality control data
- Error model (if required)
```

### Multi-Well Pad Surveys
```
Anti-Collision Data:
- All wells on pad
- Adjacent lease wells
- Separation distance verification
- Traveling cylinder calculations
```

## Regulatory Inspection Reports

### Self-Inspection Records
```
Keep Records of:
- BOP inspections (daily visual)
- BOP tests (pressure/function)
- Safety equipment checks
- Environmental inspections
- Pollution prevention inspections
```

### Agency Inspection Response
```
Following Agency Inspection:
- Review findings with inspector
- Sign inspection report
- Note any disagreements
- Submit corrective action plan (if INC issued)
- Document corrections made
- Request re-inspection if needed
```

```

- Drilling permits
- Completion reports
- Production reports
- Organization reports

Oklahoma: OCC OKAP System
- Permits
- Reports
- Well data

New Mexico: OCD Permitting
- C-forms
- Production data

North Dakota: NDIC OASIS
- Permits
- Reports
- Well files
```

### Federal Electronic Systems
```
BSEE eWell: Offshore well reporting
BOEM TIMS: Lease management
BLM AFMSS: Onshore federal
```

## Document Retention

### Minimum Retention Periods
```
Drilling Records: 5-7 years after P&A
Production Records: 5-7 years
Well Logs: Life of well + 5 years
Safety Records: 5 years minimum
Environmental Records: 5-30 years (varies)
Incident Reports: 5 years minimum
Personnel Records: Duration of employment + 3 years
Training Records: 5 years
```

### Critical Documents to Retain
```
- APD and approval
- Drilling reports
- Completion report
- Cement records
- BOP test records
- Survey records
- Well logs
- Production records
- Incident reports
- Correspondence with agencies
```
