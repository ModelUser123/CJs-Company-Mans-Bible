## Inspection Criteria

### Drill Pipe Inspection Categories

| Category | Inspection Method | Criteria |
|----------|------------------|----------|
| Visual | Direct observation | Cracks, pitting, wear |
| Dimensional | Measurement | Wall thickness, OD/ID |
| EMI | Electromagnetic | Wall loss, pitting |
| Ultrasonic | UT thickness | Remaining wall |
| MPI | Magnetic particle | Surface cracks |
| Hardbanding | Visual/UT | Condition, thickness |

### Critical Dimensions

```
TOOL JOINT INSPECTION
├── Box
│   ├── ID wear (max +1/32")
│   ├── Counterbore depth
│   ├── Shoulder flatness
│   └── Thread condition
├── Pin
│   ├── OD wear (max -1/32")
│   ├── Bevel diameter
│   ├── Pin length
│   └── Thread condition
└── Both
    ├── Hardbanding condition
    ├── Tong space
    └── Make-up torque marks
```

### Rejection Criteria

| Component | Defect | Reject When |
|-----------|--------|-------------|
| Tube | Wall loss | <80% new (Class 1) |
| Tube | Pitting | >1/16" deep |
| TJ Box | ID wear | >1/32" oversize |
| TJ Pin | OD wear | >1/32" undersize |
| Threads | Galling | >5% area |
| Threads | Stretch | Visible |
| Hardbanding | Cracked | Any cracks |

## Drill String Design Considerations

### Weight on Bit Calculation

```
Available WOB = Buoyed DC Weight - (Safety Factor × 15%)

Buoyed Weight = Air Weight × Buoyancy Factor
Buoyancy Factor = (65.5 - MW) / 65.5

Example:
20 DCs × 31 ft × 100 lb/ft = 62,000 lb air weight
BF = (65.5 - 12) / 65.5 = 0.817
Buoyed = 62,000 × 0.817 = 50,654 lb
Available WOB = 50,654 × 0.85 = 43,056 lb
```

### Neutral Point Location

```
Neutral Point = Point where buoyed weight = tension

Below neutral point: Compression (buckling possible)
Above neutral point: Tension

Keep neutral point in drill collars, not drill pipe
```

### Drill String Fatigue

| Factor | High Fatigue | Low Fatigue |
|--------|-------------|-------------|
| Dogleg | >5°/100 ft | <3°/100 ft |
| Rotation | >150 RPM | <100 RPM |
| Tension variation | High | Low |
| Pipe class | Class 2/3 | Premium |
