## Quick Reference

### Rig Classification by Hook Load

| Class | Hook Load (lb) | Typical Application |
|-------|---------------|---------------------|
| 250 | 250,000 | Shallow wells <8,000 ft |
| 350 | 350,000 | Medium wells 8,000-12,000 ft |
| 500 | 500,000 | Deep wells 12,000-18,000 ft |
| 750 | 750,000 | Deep/ERD wells 18,000-25,000 ft |
| 1000 | 1,000,000 | Ultra-deep/ERD 25,000+ ft |
| 1500 | 1,500,000 | Extreme ERD, offshore deepwater |

### Mud Pump Classification

| Type | Typical HP | Max Pressure | Max Flow |
|------|-----------|--------------|----------|
| Light Duty | 500-1000 | 3,500 psi | 400 gpm |
| Medium | 1000-1600 | 5,000 psi | 600 gpm |
| Heavy Duty | 1600-2200 | 7,500 psi | 800 gpm |
| Ultra HD | 2200-3000 | 10,000 psi | 1000 gpm |

### BOP Stack Configurations

| Configuration | Components | Typical Pressure Rating |
|--------------|------------|------------------------|
| Single RAM | 1 pipe ram, 1 blind ram | 5,000 psi |
| Double RAM | 2 pipe rams, 1 blind/shear | 10,000 psi |
| Triple RAM | 3 rams (pipe, blind, shear) | 15,000 psi |
| Quad RAM | 4 rams + annular | 15,000-20,000 psi |
| Subsea | 5+ rams, 2 annulars, connectors | 15,000-20,000 psi |

## Equipment Maintenance Philosophy

### Preventive Maintenance Tiers

```
TIER 1 - DAILY
├── Visual inspections
├── Fluid level checks
├── Safety equipment check
└── Operational tests

TIER 2 - WEEKLY
├── BOP function tests
├── Accumulator tests
├── Detailed inspections
└── Lubrication

TIER 3 - MONTHLY/RIG MOVE
├── Complete equipment inspection
├── Pressure testing
├── Wear measurement
└── Component replacement as needed

TIER 4 - ANNUAL/MAJOR
├── Third-party inspection
├── Recertification
├── Major component overhaul
└── Load testing
```
