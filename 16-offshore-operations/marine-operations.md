# Marine Operations

## Vessel Types and Operations

### Supply Vessels

```
PLATFORM SUPPLY VESSEL (PSV)
├── Length: 200-300 ft
├── Deck Space: 5,000-10,000 sq ft

├── Functions
│   ├── Anchor handling

│   ├── Towing
│   └── Supply duties
├── Equipment
│   ├── Anchor winch
│   ├── Shark jaws

FAST SUPPLY VESSEL (FSV)
├── Length: 150-250 ft
├── Speed: 20-35 knots
├── Functions
│   ├── Time-critical cargo
│   ├── Personnel transfer
│   └── Light supply
└── Deck Space: 2,000-5,000 sq ft

CREW BOATS
├── Length: 100-200 ft
├── Passengers: 50-150
├── Speed: 20-35 knots
└── Functions
    ├── Crew change
    └── Light cargo
```

## Helicopter Operations

### Helideck Requirements

| Requirement | Specification |
|-------------|--------------|
| Size | 1.25× helicopter rotor diameter |
| Load capacity | Max takeoff weight + 50% |
| Fire suppression | Foam system required |
| Lighting | Perimeter, flood, obstruction |
| Wind indicators | Sock or tetrahedron |
| Nets | Safety nets on perimeter |

### Flight Operations

```
PASSENGER OPERATIONS
├── Safety Briefing
│   ├── Emergency exits
│   ├── Life jacket location
│   ├── Ditching procedure
│   └── EBS (Emergency Breathing)
├── Weight Limits
│   ├── Passenger + baggage: 300 lb typical
│   ├── Baggage: 40 lb typical
│   └── No hazmat in cabin
└── Manifest Requirements
    ├── Full name
    ├── Employer
    ├── Weight (with bag)
    └── Destination

WEATHER MINIMUMS
├── Visibility: 1 mile typical
├── Cloud Base: 500 ft typical
├── Wind: 50 knots max (varies)
├── Sea State: Company specific
└── Night Operations: Enhanced requirements
```

### Helideck Procedures

| Phase | Actions |
|-------|---------|
| Pre-arrival | Clear deck, FOD check, fire team standby |
| Approach | Radio contact, permission to land |
| Landing | Fire team positioned, no movement |
| Passengers | Exit with head down, baggage clear |
| Departure | Clear deck, permission for takeoff |
| Post-flight | Log flight, reset equipment |

## Crane Operations

### Offshore Crane Specifications

| Rating | Capacity | Boom Length | Application |
|--------|----------|-------------|-------------|
| Light | 10-20 ton | 60-80 ft | Light lifts, PSV |
| Medium | 35-60 ton | 80-120 ft | Standard operations |
| Heavy | 100-200 ton | 120-180 ft | Heavy equipment |
| Subsea | 200-400 ton | 200+ ft | Subsea, module |

### Crane Operating Limits

```
WEATHER LIMITATIONS
├── Wind Speed
│   ├── Max operation: 25-35 knots (varies)
│   ├── Reduced capacity: 20-25 knots
│   └── Warning: >15 knots sustained
├── Vessel Motion (offloading)
│   ├── Heave: <3 ft
│   ├── Pitch: <3°
│   └── Roll: <3°
└── Visibility
    └── Minimum: 0.5 mile

LOAD LIMITATIONS
├── SWL = Marked capacity × De-rating
├── De-rating Factors
│   ├── Boom angle
│   ├── Boom length
│   ├── Vessel motion
│   └── Wind load
└── Never exceed 100% SWL
```

## Weather Management

### Sea State Scale

| Douglas Scale | Wave Height (ft) | Description | Operations |
|--------------|-----------------|-------------|------------|
| 0 | 0 | Calm | All ops |
| 1 | 0-0.3 | Rippled | All ops |
| 2 | 0.3-1.6 | Smooth | All ops |
| 3 | 1.6-4 | Slight | All ops |
| 4 | 4-8 | Moderate | Limited crane |
| 5 | 8-13 | Rough | No boat transfer |
| 6 | 13-20 | Very rough | Suspend crane |
| 7 | 20-30 | High | Emergency only |
| 8 | 30-45 | Very high | Evacuate |
| 9 | >45 | Phenomenal | Survive |

### Weather Decision Matrix

| Parameter | Green | Yellow | Red |
|-----------|-------|--------|-----|
| Wind (kts) | <25 | 25-40 | >40 |
| Wave (ft) | <8 | 8-15 | >15 |
| Visibility | >3 mi | 1-3 mi | <1 mi |
| Lightning | >10 mi | 5-10 mi | <5 mi |
| Helicopter | Normal | Enhanced | Suspended |
| Crane | Normal | Reduced | Suspended |
| Boat transfer | Normal | Caution | Suspended |

### Hurricane Procedures

```
MONITORING PHASE (120+ hours)
├── Track storm daily
├── Review hurricane procedures
├── Inventory emergency supplies
└── Coordinate with logistics

PREPARATION PHASE (72 hours)
├── Begin well operations for secure point
├── Arrange evacuation transportation
├── Reduce POB (non-essential)
└── Top off consumables

SECURE PHASE (48 hours)
├── Complete well securing
├── Set subsea tree/hurricane hanger
├── Secure all loose equipment
├── Begin non-essential evacuation

EVACUATION PHASE (24 hours)
├── Evacuate ALL personnel
├── Final equipment securing
├── Activate emergency beacons
├── Document rig condition

POST-STORM
├── Damage assessment
├── Equipment testing
├── Regulatory notification
└── Resume operations when safe
```

## Anchor Handling

### Mooring System Components

```
8-POINT MOORING SYSTEM
├── Anchors (8)
│   ├── Stevpris/Vryhof type
│   ├── 15-25 ton each
│   └── Penetration tested
├── Anchor Chain
│   ├── 3-4" diameter
│   ├── Grade 3/R3S/R4
│   └── 2,000-3,000 ft/leg
├── Wire Rope (optional)
│   ├── Reduces weight
│   ├── 3-4" diameter
│   └── Polyester insert section
└── Windlasses
    ├── 8 units
    ├── Tension monitoring
    └── Auto-tension capability

12-POINT MOORING
├── For harsh environments
├── Greater holding capacity
└── More complex operations
```
