# Sand Control

## Overview

Sand control prevents formation sand from entering the wellbore, protecting equipment and maintaining productivity. Selecting the right sand control method is critical for wells in unconsolidated formations.

---

## Why Sand Control?

### Problems from Sand Production

| Problem | Consequence |
|---------|-------------|
| **Erosion** | Equipment damage, failures |
| **Fill** | Tubing/casing fill, reduced production |
| **Surface handling** | Separator issues, disposal |
| **Subsidence** | Formation collapse, casing damage |
| **Environmental** | Sand disposal requirements |
| **Safety** | Erosion of safety equipment |

### When Sand Control is Needed

| Indicator | Method |
|-----------|--------|
| **Offset well history** | Previous wells produced sand |
| **Formation strength** | UCS < 1,000 psi typically |
| **Core analysis** | Unconsolidated samples |
| **Log analysis** | Sonic indicates weak rock |
| **Drawdown analysis** | High drawdown causes failure |

## Sand Control Methods

### Comparison of Methods

| Method | Mechanism | Application |
|--------|-----------|-------------|
| **Gravel pack** | Physical filter | High perm, unconsolidated |
| **Frac pack** | Gravel in fracture | Moderate perm, damage |
| **Standalone screen** | Screen exclusion | Lower risk, simpler |
| **Expandable screen** | Conform to wellbore | Open hole, horizontal |
| **Chemical consolidation** | Bind sand grains | Small intervals |
| **Rate/pressure control** | Manage drawdown | Low sand tendency |

## Gravel Pack

### Concept

```
┌─────────────────────────────────────────────┐
│              CASED HOLE GRAVEL PACK         │
├─────────────────────────────────────────────┤
│                                             │
│   Casing ──────│  │                         │
│                │  │                         │
│   Cement ──────│██│                         │
│                │  │                         │
│   Perforations ●●●│  │●●●                   │
│                │  ╠══╣  │ ← Screen          │
│   Gravel ──────│▓▓│  │▓▓│                   │
│                │  │  │  │                   │
│   Screen ──────│  ╠══╣  │                   │
│                │▓▓│  │▓▓│                   │
│   Perforations ●●●│  │●●●                   │
│                │  │  │                       │
│                │██│                         │
│                │  │                         │
└─────────────────────────────────────────────┘

Legend: ██ = Cement  ▓▓ = Gravel  ●●● = Perforations
```

### Gravel Sizing

**Saucier Criteria (Industry Standard)**:
```
Gravel Size = 5-6 × D50 of formation sand

Where D50 = Median grain size of formation
```

**Common Gravel Sizes**:

| Gravel Size | Formation D50 Range |
|-------------|---------------------|
| 12/20 mesh | Very coarse sand |
| 20/40 mesh | Coarse to medium sand |
| 40/60 mesh | Medium to fine sand |
| 50/70 mesh | Fine sand |

### Gravel Pack Execution

**Steps**:
1. Perforate (high shot density, big hole)
2. Clean perforations (acid, wash)
3. Run screen and packer assembly
4. Position tools, set packer
5. Pump gravel slurry
6. Place gravel in perforations and annulus
7. Reverse out excess
8. Test and complete

### Gravel Pack Fluid Systems

| Type | Viscosifier | Application |
|------|-------------|-------------|
| **Brine** | HEC polymer | Most common |
| **Crosslinked gel** | Borate-crosslinked | Long intervals, deviated |
| **Viscoelastic** | Surfactant-based | Clean breaks |

## Frac Pack

### Concept

Combines hydraulic fracturing with gravel packing:
- Creates fracture to bypass near-wellbore damage
- Places proppant in fracture
- Packs annulus with gravel
- Higher productivity than conventional GP

### When to Use Frac Pack

| Factor | Frac Pack Advantage |
|--------|---------------------|
| **Near-wellbore damage** | Bypasses damage |
| **Low permeability** | Improves PI |
| **Long intervals** | Better coverage |
| **Complex geology** | Overcomes heterogeneity |

### Frac Pack Execution

| Step | Action |
|------|--------|
| 1 | Perforate with high shot density |
| 2 | Run screen and service tools |
| 3 | Perform mini-frac (calibration) |
| 4 | Pump proppant-laden frac fluid |
| 5 | Screen out at designed pressure |
| 6 | Pack annulus with remaining proppant |
| 7 | Reverse out excess |

## Standalone Screens

### Screen Types

| Type | Description |
|------|-------------|
| **Wire-wrapped** | Wire wound on base pipe, slots |
| **Pre-packed** | Gravel layer around screen |
| **Premium mesh** | Multiple filtration layers |
| **Sintered metal** | Fused metal particles |

### Screen Selection

| Factor | Consideration |
|--------|---------------|
| **Slot/pore size** | Match to formation size |
| **OD** | Annular clearance |
| **ID** | Flow capacity |
| **Strength** | Collapse, tensile |
| **Length** | Cover pay interval |

### Slot Sizing

**Rule of Thumb**:
```
Slot Size = 2 × D10 of formation sand

Where D10 = Size at which 10% is finer
```

### Standalone Screen Applications

| Application | Suitability |
|-------------|-------------|
| **Open hole horizontal** | Common, good option |
| **Moderate sand risk** | May be adequate |
| **Low budget** | Lower cost option |
| **Moderate rates** | Can be limiting at high rates |

## Expandable Screens

### Concept

- Run into wellbore in collapsed state
- Expanded to contact formation
- Provides mechanical support
- Improves formation stability

### Advantages

| Advantage | Benefit |
|-----------|---------|
| **Formation contact** | Supports sand |
| **Larger ID** | Better productivity |
| **Full hole coverage** | No annular gap |
| **Open hole capable** | Horizontal wells |

## Chemical Consolidation

### Methods

| Method | Mechanism |
|--------|-----------|
| **Resin** | Injected resin binds grains |
| **Plastic consolidation** | Chemical reaction |
| **Zeta potential modification** | Fines stabilization |

### Application

- Small intervals
- Touch-up after other methods
- Low sand volumes
- Not primary method for major issues

## Sand Monitoring

### Production Monitoring

| Method | Measurement |
|--------|-------------|
| **Sand probes** | Erosion/acoustic detection |
| **Separator sampling** | Visual/weight |
| **Acoustic monitors** | Sand impact sound |
| **Erosion coupons** | Long-term trend |

### Acceptable Sand Rates

| Rate | Status |
|------|--------|
| < 1 lb/1000 bbl | Acceptable |
| 1-5 lb/1000 bbl | Monitor closely |
| 5-50 lb/1000 bbl | Intervention likely needed |
| > 50 lb/1000 bbl | Critical, immediate action |

## Sand Control Selection

### Decision Factors

| Factor | Method Tendency |
|--------|-----------------|
| **High perm (>500 md)** | Gravel pack, standalone |
| **Moderate perm (50-500 md)** | Frac pack, gravel pack |
| **Low perm (<50 md)** | Frac pack |
| **Horizontal well** | Standalone, expandable |
| **Severe sand** | Gravel pack, frac pack |
| **Cost-sensitive** | Standalone, rate control |

### Selection Flowchart

```
                Start
                  │
                  ▼
         Severe sand risk?
             /        \
           Yes         No
            │           │
            ▼           ▼
    Deep damage?    Standalone
     or low perm?    Screen
      /     \
    Yes      No
     │        │
     ▼        ▼
  Frac      Gravel
  Pack       Pack
```

## Troubleshooting

### Common Problems

| Problem | Cause | Solution |
|---------|-------|----------|
| **Screen plugging** | Fines migration | Chemical treatment, drawdown control |
| **Pack void** | Incomplete packing | Re-pack, squeeze |
| **Screen erosion** | High velocity, poor pack | Reduce rate, repack |
| **Sand bypassing** | Screen damage, sizing wrong | Intervention, recompletion |
| **Productivity loss** | Damage, plugging | Acid, re-stimulate |
