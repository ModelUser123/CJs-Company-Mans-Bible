# Packer Selection

## Overview

Packers provide the seal between tubing and casing, isolating production from the annulus. Proper packer selection is critical for well integrity, safety, and long-term performance.

---

## Packer Functions

### Primary Functions
1. **Seal annulus** - Prevent flow between tubing and casing
2. **Anchor tubing** - Support tubing loads
3. **Zone isolation** - Separate producing intervals
4. **Protect casing** - Shield from well fluids
5. **Enable operations** - Support stimulation, testing

---

## Packer Types

### Retrievable Packers

**Description**: Can be pulled from well

| Advantage | Disadvantage |
|-----------|--------------|
| Can be retrieved for workover | Generally lower pressure rating |
| Tubing can be pulled with packer | More complex design |
| Reusable | Larger OD for same ID |
| Easier fishing if stuck | |

**Applications**:
- Exploration and appraisal wells
- Wells requiring frequent workover
- Single-zone completions
- Temporary completions

### Permanent Packers

**Description**: Designed to remain in well for life

| Advantage | Disadvantage |
|-----------|--------------|
| Higher pressure ratings | Cannot be retrieved |
| Smaller OD | Must be milled to remove |
| Simpler design | More commitment required |
| More reliable long-term | Tubing pulled, leaving packer |
| Lower cost | |

**Applications**:
- Development wells
- High-pressure differentials
- Multi-zone completions
- Hostile environments

### Production Packers

| Type | Setting Method |
|------|----------------|
| **Mechanical** | Set by tubing rotation and/or slackoff |
| **Hydraulic** | Set by internal pressure |
| **Wireline** | Set by wireline tools |
| **Electric line** | Set by electrical signal |

## Setting Mechanisms

### Mechanical Set

```
┌─────────────────────────────────────────┐
│           MECHANICAL SET SEQUENCE       │
├─────────────────────────────────────────┤
│                                         │
│   1. Run to depth on tubing             │
│              │                          │
│              ▼                          │
│   2. Apply rotation (right-hand)        │
│      Releases J-slot mechanism          │
│              │                          │
│              ▼                          │
│   3. Slack off weight                   │
│      Compresses packer elements         │
│              │                          │
│              ▼                          │
│   4. Pick up weight to confirm set      │
│      Overpull indicates set             │
│                                         │
└─────────────────────────────────────────┘
```

### Hydraulic Set

| Step | Action |
|------|--------|
| 1 | Run to depth |
| 2 | Apply internal pressure |
| 3 | Shear setting device |
| 4 | Pressure expands elements |
| 5 | Slips engage casing |

### Wireline Set

- Run on electric line or slickline
- Setting tool fires charge or applies force
- Packer expands and locks
- Setting tool releases

## Packer Specifications

### Key Rating Parameters

| Parameter | Definition |
|-----------|------------|
| **Differential pressure** | Max pressure across packer |
| **Collapse rating** | Max external pressure |
| **Burst rating** | Max internal pressure |
| **Tensile load** | Max pulling force |
| **Compression load** | Max set-down weight |
| **Temperature rating** | Max operating temperature |

### Typical Ratings

| Packer Type | Differential (psi) | Temp (°F) |
|-------------|-------------------|-----------|
| **Retrievable** | 5,000-10,000 | 300-400 |
| **Permanent** | 10,000-15,000 | 350-450 |
| **High-spec permanent** | 15,000-20,000 | 400-500 |

## Bore Sizes and Clearances

### Packer Bore Selection

| Consideration | Impact |
|---------------|--------|
| **Tubing size** | Must match or accept tubing |
| **Through-bore tools** | Size of intervention tools |
| **Flow rate** | Larger bore = less restriction |
| **Pump capability** | ESP, rod pump clearances |

## Packer Selection Process

### Step 1: Define Requirements

| Factor | Questions |
|--------|-----------|
| **Well conditions** | Pressure, temperature, depth? |
| **Fluids** | H2S, CO2, corrosive? |
| **Life expectancy** | How long must it last? |
| **Operations** | Workover access needed? |
| **Zonal requirements** | How many zones? |

### Step 2: Evaluate Options

| Criterion | Retrievable | Permanent |
|-----------|-------------|-----------|
| Initial cost | Higher | Lower |
| Workover cost | Lower | Higher |
| Pressure rating | Lower | Higher |
| Reliability | Good | Excellent |
| Flexibility | High | Low |

### Step 3: Verify Ratings

- [ ] Differential pressure adequate
- [ ] Temperature rating acceptable
- [ ] Load ratings (tension, compression) verified
- [ ] Material compatibility confirmed
- [ ] Bore size suitable
- [ ] Casing size compatible

---

## Packer Running Procedures

### Pre-Run Checks

- [ ] Verify packer specifications
- [ ] Confirm casing ID (caliper if needed)
- [ ] Check setting tool
- [ ] Verify depth correlation method
- [ ] Review setting procedure
- [ ] Confirm completion fluid weight

### Setting Depth Selection

| Factor | Consideration |
|--------|---------------|
| **Casing condition** | Good cement, no damage |
| **Above perforations** | Typically 50-200 ft |
| **Collar location** | Avoid setting on collar |
| **Cement top** | Above production cement |
| **Future operations** | Allow tool access |

### Setting Verification

| Check | Method |
|-------|--------|
| **Weight indication** | Overpull shows set |
| **Pressure test** | Apply annular pressure |
| **Circulation** | Unable to circulate |
| **Slickline** | Tag packer |

## Packer Problems and Solutions

### Common Problems

| Problem | Cause | Solution |
|---------|-------|----------|
| **Won't set** | Wrong weight, depth, worn slip | Re-position, apply more weight |
| **Leaking** | Element damage, casing corrosion | Reset, repair casing |
| **Stuck** | Debris, corrosion, differential | Work pipe, pump chemicals |
| **Can't release** | Mechanism failure | Milling required |

### Packer Removal (Permanent)

| Method | Description |
|--------|-------------|
| **Milling** | Cut away with mill |
| **Section milling** | Remove with casing section |
| **Chemical cutting** | Use chemical cutter |
| **Washover** | Wash over with larger string |

## Special Packer Applications

### Gravel Pack Packers

- Large ID for gravel flow
- Isolation above pack
- Pressure rating for squeezing

### Frac Packers

- High pressure rating (10,000-15,000 psi)
- High temperature capability
- Large bore for high rates

### ESP Packers

- Large bore for ESP assembly
- Cable pass-through
- Gas separation capability
